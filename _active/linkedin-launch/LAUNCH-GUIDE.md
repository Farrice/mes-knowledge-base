## Execution Checklist

### Phase 1: Setup (Do Once)

- [ ] **Host prompt kits** — Copy the XML block from each `prompt-kits/0X-*.md` into a Notion public page or Google Doc (one per kit)
- [ ] **Save hosted links** — Record each prompt kit URL for DM delivery
- [ ] **Profile check** — Update LinkedIn headline, banner, and About section to match positioning (ghostwriting for coaches)
- [ ] **DM scripts ready** — Review `outreach/dm-scripts.md` and customize any personal details

### Phase 2: Post Schedule (5 Posts)

Post Tue-Thu, 8-10am EST. One post per publishing day.

| # | Post | CTA Keyword | Prompt Kit to Send | File |
|---|------|------------|-------------------|------|
| 1 | The Invisible Expert Epidemic | INVISIBLE | Invisible Expert Diagnostic | `posts/01-invisible-expert.md` |
| 2 | The 47 Google Docs Trap | DEPLOY | Deployment Gap Calculator | `posts/02-47-google-docs-trap.md` |
| 3 | The Referral Ceiling | PIPELINE | Pipeline Fragility Score | `posts/03-referral-ceiling.md` |
| 4 | Your Clients Already Wrote Your Content | MINE | Client Language Miner | `posts/04-clients-wrote-your-content.md` |
| 5 | The 30-Minute Lie | VOICE | Voice Match Scorecard | `posts/05-30-minute-lie.md` |

### Phase 3: Per-Post Execution (Repeat for Each Post)

1. **Copy post** from `posts/0X-*.md` → paste into LinkedIn editor
2. **Publish** during Tue-Thu, 8-10am EST window
3. **First comment** — Drop the hosted prompt kit link immediately after publishing
4. **Monitor comments** — When someone drops the CTA keyword, DM them using the script from `outreach/dm-scripts.md`
5. **DM delivery** — Send the hosted prompt kit link + warm follow-up
6. **Conversion** — After they run the kit, follow up with Proof Run offer from `offers/proof-run-offer-page.md`

### Phase 4: Conversion Flow

```
Post → Keyword comment → DM prompt kit → They run it → Follow-up DM → Proof Run ($500)
```

- **Proof Run details**: `offers/proof-run-offer-page.md`
- **Delivery process**: `offers/proof-run-delivery-template.md`
- **Target**: $500 one-time → demonstrate value → convert to monthly retainer

### Phase 5: Scale (After First 5 Posts)

- **13 more concepts ready** in `mini-briefs/concept-dossier.md`
- Run `/mini-brief` on any concept to produce the next production-ready package
- Chain: `/flywheel-ideas` → `/mini-brief` → `/ip-flywheel` → publish
