### The AI Ghostwriting Workflow:

#### Phase 1: Voice Extraction (Week 1)
1. **Collect 15-20 writing samples** from client
2. **Create Voice Profile** using AI analysis
3. **Document**: Phrases used/avoided, sentence structure, tone, pacing
4. **Build Custom GPT** trained on client voice

#### Phase 2: Content Mining (Monthly)
1. **Schedule 30-minute interview** with client
2. **Record and transcribe** using Fathom/Otter
3. **Extract 8-12 insights** using AI analysis
4. **Map to content calendar**

#### Phase 3: Content Creation (Weekly)
1. **AI generates first draft** using voice-trained model
2. **Human review** for voice accuracy (15 minutes)
3. **Add personal touches** — specific stories, current references
4. **Final polish** — hooks, CTAs, formatting

#### Phase 4: Quality Control (Pre-Publish)
1. **Voice consistency check** — Does this sound like the client?
2. **Fact verification** — Are all claims accurate?
3. **Scroll-stopping test** — Would I stop scrolling for this?
4. **CTA review** — Is the call-to-action clear?

### The 80/20 Balance:
- **AI handles 70-80%**: Research, transcription, first drafts, ideation
- **Humans handle 20-30%**: Voice matching, strategic editing, final polish, client relationships
