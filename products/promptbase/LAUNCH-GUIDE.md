## Submission Workflow (Per Prompt)

1. Go to https://promptbase.com/sell
2. Open the markdown file for the prompt you're submitting
3. Copy each section into the matching form field:

| File Section | PromptBase Form Field |
|-------------|----------------------|
| **TITLE** | Title field |
| **DESCRIPTION** | Description field |
| **TAGS** | Tags field |
| **PROMPT TEMPLATE** | Prompt field |

| **EXAMPLE OUTPUT 3** | Example Output 3 field |
| **EXAMPLE OUTPUT 4** | Example Output 4 field |
