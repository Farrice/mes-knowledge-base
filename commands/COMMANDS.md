## Agent Deployment

### Deploy Single Agent
```
deploy @agent-name for [task]
```

**Examples:**
- `deploy @copywriter-supreme for my webinar registration page`
- `deploy @viral-engineer for my TikTok strategy`
- `deploy @sales-closer for my discovery calls`

### Convene Agent Council
```
convene @agent1, @agent2, @agent3 for [project]
```

**Examples:**
- `convene @brand-strategist, @copywriter-supreme, @growth-hacker for my course launch`
- `convene @offer-architect, @sales-closer, @proposal-writer for my consulting business`

### Agent Handoff
```
start with @agent1 then @agent2 for [project]
```

**Example:**
- `start with @persona-profiler then @copywriter-supreme for my sales page`

## Modifier Commands

Add these to any command for specific behavior:

| Modifier | Effect |
|----------|--------|
| `--deep` | Maximum depth analysis |
| `--quick` | Fast, actionable output |
| `--examples` | Include real examples |
| `--variations` | Multiple options/approaches |
| `--critique` | Include self-analysis |
| `--compare` | Compare approaches |

**Examples:**
- `hooks for my coaching offer --variations` → Get 3 different hook styles
- `sales page for my course --deep` → Comprehensive long-form page
- `offer design --compare` → Compare multiple offer structures
