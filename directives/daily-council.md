# Directive: Daily Council

> Morning routine leveraging your expert council for daily focus

---

## Purpose

Start each day with strategic input from your expert council. Get clarity on priorities, overcome blocks, and set intention for high-impact work.

---

## When to Use

- Morning planning sessions
- Weekly review and planning
- When feeling scattered or unfocused
- Starting new projects or initiatives

---

## Daily Council Routine (15-20 minutes)

### Phase 1: Situation Scan (3 minutes)
Briefly describe your current state:
- What's the most important thing to accomplish today?
- What's blocking you or creating resistance?
- What decision is weighing on you?

### Phase 2: Expert Consultation (10 minutes)

**For Focus/Priority Questions:**
Ask Jim O'Shaughnessy for strategic synthesis:
```
@jim-oshaughnessy Given [situation], what should I prioritize today?
```

**For Identity/Mindset Blocks:**
Ask Michael Bernoff:
```
@michael-bernoff I'm feeling [resistance/block]. Help me reframe.
```

**For Content/Creation Days:**
Ask Shaan Puri:
```
@shaan-puri What's the highest-leverage content I could create today?
```

**For Sales/Revenue Days:**
Ask Jeremy Miner:
```
@jeremy-miner I have [conversation/pitch] today. What's the frame?
```

**For Brand/Positioning Work:**
Ask Dai Media:
```
@dai-media Help me see my consumer clearly for today's work.
```

### Phase 3: Capture & Commit (5 minutes)
- Write down the key insight from your consultation
- Define your #1 priority for the day
- Identify one specific action to take first
- Set an intention statement
