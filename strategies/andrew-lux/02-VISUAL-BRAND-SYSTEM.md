# Andrew Lux: Visual Brand System

## Current State

Andrew's positioning: **"Coastal energy" + "LA Palm Trees & Disco Balls"**

This is good instinct—it's specific to place, evokes a vibe, and differentiates from the typical "dark warehouse techno" or "generic EDM festival" aesthetic.

But it needs to be systematized into a repeatable, recognizable visual language.

---

## The Visual Identity Framework

### The 5 Laws of DJ Visual Branding

1. **Own One Thing** — deadmau5 owns the mau5head. Rezz owns spiraling LED goggles. What will Andrew own?

2. **Sound = Visual** — Your aesthetic must feel like your music sounds.

3. **Simplicity Scales** — If a fan can't draw your logo from memory, it's too complex.

4. **Consistency Compounds** — Same colors, fonts, symbols EVERYWHERE builds recognition faster than any marketing budget.

5. **Experience > Object** — Eventually, your shows become your brand.

## Andrew's Visual Direction

### The "Coastal Bass" Aesthetic

**Concept:** California's golden hour meets bass house energy. Sunset gradients that fade into neon nightlife. The warm daytime vibe that turns into late-night festival chaos.

**Why it works:**
- Matches his sound (warm but hard-hitting)
- Matches his story (LA journey)
- Unique in bass house (most go dark/aggressive)
- Instantly recognizable color palette

### Core Visual Motifs

| Element | Description |
|---------|-------------|
| **Sunset gradients** | Orange → pink → purple transitions |
| **Palm silhouettes** | LA iconography, works at any scale |
| **Ocean textures** | Waves, water reflections, beach |
| **Neon accents** | Night club/festival energy |
| **Golden hour light** | Warm, inviting, aspirational |

### Color Palette

**Primary Colors:**
- **Sunset Orange** (#FF6B35) — Warmth, energy
- **Pacific Blue** (#0077B6) — Ocean, California coast
- **Dusk Purple** (#7B2D8E) — Night transition, EDM energy

**Secondary Colors:**
- **Sand/Cream** (#F5E6D3) — Neutral base
- **Deep Navy** (#1A1A2E) — Dark base for contrast
- **Hot Pink** (#FF1493) — Accent for high energy moments

**Gradient Application:**
- Social posts: Sunset gradient backgrounds
- Cover art: Orange → purple transitions
- Stage visuals: Blue → pink → orange flows

### Typography

**Primary Font:** Clean, modern sans-serif (think Montserrat, Bebas Neue, or custom)
- Used for: Name, headings, announcements

**Secondary Font:** Slight retro/80s influence (think Miami Vice meets modern)
- Used for: Accents, dates, supporting text

**Rules:**
- Always bold for name/logo
- High contrast against background
- Consistent weight and spacing across all uses

## The Signature Visual Element

### Option 1: The "Coastal Lens" Effect

A distinctive visual treatment applied to all content:
- Warm color grading (golden hour filter)
- Slight lens flare effects
- Palm tree framing
- Consistent preset across all posts

**Pros:** Easy to apply, consistent without requiring design skills
**Cons:** Subtle, not as instantly ownable as a symbol

### Option 2: The "LUX" Mark

A stylized version of his name that becomes the logo:
- "LUX" in custom typography
- Integrated with sunset/wave visual element
- Simple enough to draw, distinctive enough to remember

**Pros:** Name recognition, works on merch
**Cons:** Requires professional design investment

### Option 3: The Palm Wave

A combined palm tree + sound wave symbol:
- Palm tree silhouette morphing into an audio waveform
- Captures both "coastal" and "music"
- Can be used as standalone mark or with name

**Pros:** Unique, ownable, tells the brand story
**Cons:** More complex, needs careful execution

### Recommendation

Start with **Option 1 (Coastal Lens effect)** for immediate consistency. Invest in **Option 2 or 3** when budget allows for professional logo design.

**Immediate action:** Create a Lightroom/VSCO preset that applies the "Coastal" look to all content. Use it on EVERYTHING.

## Platform-Specific Guidelines

### Instagram

**Grid Aesthetic:**
- Alternate between performance shots, lifestyle, and promo graphics
- Every 3rd post should be more colorful/graphic to create visual rhythm
- Stories: Use consistent highlight covers (sunset gradient)

**Content types:**
- Performance clips: Warm color grading, show energy
- Lifestyle: California aesthetic (beach, palm trees, golden hour)
- Promos: Use brand colors consistently
- Reels: Same treatment as TikTok content

### TikTok

**Visual Treatment:**
- Same color grading as Instagram
- Text overlays in brand fonts/colors
- Show hands on decks (authenticity)
- Coastal locations when filming in person

### YouTube

**Thumbnails:**
- Consistent template with brand colors
- Face visible (builds recognition)
- High contrast for visibility at small sizes

**Channel Art:**
- Sunset gradient background
- Clear logo placement
- Show schedule if consistent

### Cover Art (Spotify, Beatport, etc.)

**The 2025 Trend:** Authenticity over AI gloss. Analog textures, noise overlays, human-centric.

**Andrew's Approach:**
- Sunset/coastal imagery
- Subtle noise textures
- Typography that matches brand
- Consistent color palette across all releases

**Template approach:** Create a system where each release feels connected but distinct. Same fonts, same color family, different specific images.

## Merch Design Direction

### Core Principles

1. **Would I wear this?** — Merch should be fashion, not advertisement
2. **Subtle > loud** — Small logo placement, quality materials
3. **Coastal vibe** — Colors and imagery that match the brand
4. **Limited drops** — Scarcity creates value

### Product Ideas

| Item | Design Direction |
|------|-----------------|
| **T-shirt (minimal)** | Small "LUX" logo left chest, sunset gradient back print |
| **Hoodie** | Embroidered palm tree + wave mark |
| **Dad hat** | Simple "LUX" embroidery, sunset color options |
| **Beach towel** | Full sunset gradient, brand name |
| **USB drive** | Custom shape (surfboard? palm tree?), loaded with exclusive mix |

### Design Rules

- Same logo treatment everywhere
- Same color palette
- High quality printing/embroidery
- If it doesn't feel like the brand, don't make it

## The Consistency Test

Ask before posting anything:

1. **Does this feel like Andrew Lux?** (Coastal, warm, energetic)
2. **Would I recognize this without seeing the name?** (After months of consistency)
3. **Does this match the music?** (Sound and visuals aligned)
4. **Is this consistent with everything else?** (Same colors, fonts, vibe)

If any answer is "no," don't post it. Or fix it first.
