## INPUT REQUIRED

- **[YOUR CONTENT STYLE]**: What types of content you make (silent films, talking head, lifestyle, educational, etc.)
- **[YOUR CONTENT PILLARS]**: The 3-5 topics you create content about

---

## EXECUTION PROTOCOL

1. **Analyze Content Requirements**: Based on content style and pillars, identify what types of B-roll scenes are needed most frequently. Silent film creators need more variety than talking head creators.

2. **Design Folder Architecture**: Create a logical folder structure organized by action type, emotion, and environment. Structure should enable intuitive retrievalâ€”when you need a "struggling" shot, you know exactly where to look.

3. **Generate Master Scene Categories**: List every scene type needed with specific examples of what to film. Categories should cover both literal actions and emotional states.

4. **Create Gap Analysis Framework**: Design a system to identify what's missing so filming sessions are purposeful, not random.

5. **Produce Priority Shot List**: Generate a specific, filmable shot list for an initial library-building session, prioritized by frequency of use.

6. **Deliver Complete System**: Provide folder structure, scene categories, shot list, and ongoing maintenance protocol.

- **Format**: Library architecture + actionable shot list

## CREATIVE LATITUDE

Apply full creative intelligence in identifying scene types the user hasn't considered. Often the most useful B-roll is non-obviousâ€”the "in between" moments, the transitions, the emotional states rather than literal actions.

Where you see opportunities for visual metaphors that could serve multiple content purposes, include those. A shot of hands typing could serve "working," "struggling," "building," and "late nights."

The goal is a library that makes any future content idea immediately filmable.

### GAP ANALYSIS CHECKLIST

**Weekly Review Questions**:
- [ ] Did I need a scene this week that I didn't have?
- [ ] What emotional state am I missing coverage for?
- [ ] Are my "current era" shots actually current (within 6 months)?
- [ ] Do I have both positive AND negative versions of each situation?

**Monthly Update Protocol**:
1. Review content from past 30 days
2. Note any scenes you wished you had
3. Add those to "Filming Needed" list
4. Schedule one 1-hour filming session for gaps
5. Update folder with fresh material

**Quarterly Refresh**:
- Retire outdated footage (old locations, old appearance)
- Add new environments you now frequent
- Expand emotional range based on content evolution
- Ensure minimum 20 clips per high-use category

---

### STORAGE RECOMMENDATIONS

**Organization**:
- Use cloud storage (Google Drive, Dropbox) for access across devices
- Maintain local backup on external drive
- Use consistent naming convention religiously

**File Management**:
- Keep original files in "RAW" subfolder
- Create "READY_TO_USE" versions (color-corrected, trimmed)
- Tag files with metadata if your system supports it

**Capacity Planning**:
- 1 minute of 4K = ~350MB
- Target library: 200-400 clips
- Estimated storage need: 50-150GB
