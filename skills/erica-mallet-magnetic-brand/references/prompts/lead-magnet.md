## Lead Magnet Types

**THE QUICK WIN**:
- Immediate small result
- Demonstrates methodology taste
- Minimal time investment
- "If this works, imagine the full thing"

**THE FRAMEWORK**:
- Conceptual structure
- New way of seeing problem
- Memorable naming
- Application instructions

**THE TEMPLATE/TOOL**:
- Fill-in-blank utility
- Time-saving practical asset
- Immediate use
- Results from using it

**THE CHECKLIST**:
- Comprehensive list
- Saves research time
- Quick reference
- Action-oriented

**THE CASE STUDY**:
- Proof-focused
- Story + methodology revealed
- Specific results shown
- "They did it, you can too"

**THE QUIZ/ASSESSMENT**:
- Self-discovery
- Personalized results
- Segment while engaging
- Open loop for solution

---

## Execution

### Step 1: Type Selection
Match to audience needs:
- Time-starved: Checklist, Template
- Confused: Framework, Quiz
- Skeptical: Case Study
- Action-oriented: Quick Win

### Step 2: Promise Engineering
Create irresistible offer:
- Specific outcome
- Timeframe (if applicable)
- Exclusivity angle
- Zero risk language

### Step 3: Content Architecture
Structure for value + conversion:
- Immediate value upfront
- Beliefs woven throughout
- Methodology demonstrated
- Next step positioning

### Step 4: Conversion Design
Build toward next step:
- Value delivered first
- Natural transition
- CTA feels helpful
- Urgency if authentic
