# Gap Building Engine

> "Your prospects aren't saying no to you. They're saying no because the gap between where they are compared to where they could be isn't big enough in their mind yet." — Jeremy Miner

## When to Use

- Designing sales conversations or discovery calls
- Writing landing pages or sales pages that need to create urgency
- Building diagnostic funnels that uncover hidden problems
- Creating content that motivates action (not just awareness)
- Preparing for any negotiation where the other side needs to feel urgency

## Phase 1: Current State Mapping

Document the prospect's CURRENT reality with specificity.

**Questions to extract current state:**
1. "Tell me about your current situation with [topic]..."
2. "How long has this been going on?"
3. "What have you already tried?"
4. "Are you 100% happy with [current solution/situation]?" ← 100% Framing (Pattern 27)
5. "What would you change if you could?"

**Current State Template:**
```
CURRENT STATE:
- Situation: [specific description in their words]
- Duration: [how long they've been here]
- Failed attempts: [what they've tried that didn't work]
- Emotional state: [how they feel about it]
- Ripple effects: [how it's affecting other areas]
```

**Rules:**
- Use THEIR language, not yours
- Capture emotional words verbatim
- Don't rush — verbal pacing (Pattern 28) is critical here
- Concerned tone throughout

## Phase 2: Objective State Mapping

Define where they WANT to be — their ideal outcome.

**Questions to extract objective state:**
1. "Help me understand — in a perfect world, what would [topic] look like for you?"
2. "If we could wave a magic wand, what would be different?"
3. "What would solving this actually mean for your [life/business/family]?"
4. "How would you feel if this wasn't a problem anymore?"

**Objective State Template:**
```
OBJECTIVE STATE:
- Desired outcome: [specific, vivid description]
- Emotional payoff: [how they'd feel]
- Life impact: [what changes in their broader life]
- Timeline desire: [when they want this resolved]
- Motivation: [why NOW, not later]
```

## Phase 3: Problem Discovery — The 3-Layer Drill

This is where the gap gets built. Each problem you help them discover widens the gap.

### Layer 1: Surface Problems → Real Problems

Don't accept the first answer. The first answer is always surface-level.

**Question Framework:**
- "You mentioned [surface problem]. What's caused you to feel like [current solution] might not eventually solve that?"
- "Help me understand — what do you think is REALLY going on beneath that?"
- Neutral language + doubt seeding: "[Current approach] is... fairly decent. What made you start looking at other options?"

### Layer 2: Root Cause Probing

Get them to diagnose the actual cause.

**Question Framework:**
- "What do you think is actually causing [problem they just described]?"
- "How long do you think this has been building?"
- "What happens if you trace this back — when did it really start?"

### Layer 3: Personal Impact

This is where emotion — and urgency — lives.

**Question Framework:**
- "How is that affecting you personally?"
- "How does that make you feel when you think about it?"
- "What's the toll been on your [family/health/career/peace of mind]?"
- "If nothing changes in the next [6 months / year], where does this end up?"

**Critical:** Use concerned tone (🟢) for all Layer 3 questions. Add verbal pacing — "How is that... affecting you... personally?" The pauses force internalization.

## Phase 4: Gap Amplification

Stack the problems to make the gap overwhelming.

**Amplification Technique:**
1. **Problem List Mirror:** "So let me make sure I understand... you're dealing with [Problem 1], which is caused by [Root Cause], and it's affecting your [Personal Impact]. AND you also mentioned [Problem 2] and [Problem 3]. Is there anything else you're dealing with that we haven't talked about?"

2. **Future Pain Projection:** "If all of those things continue for another [timeframe]... where do you see this going?"

3. **Consequence Stack:** Layer cascading consequences — not just "it stays bad" but "it gets WORSE because [secondary effect leads to tertiary effect]."

4. **Self-Persuasion Turn:** "Given everything you've described... what do you think needs to happen?"

## Phase 5: Bridge to Objective State

Connect the gap to your solution — but let THEM build the bridge.

**Bridge Questions:**
- "You said you want [objective state]. Given what we've uncovered, what would put you in the best position to actually get there?"
- "What would it mean for your [family/business/life] if you could solve these [X] problems in the next [timeframe]?"

## Output Format

For any given context, deliver:

```
CURRENT STATE MAP:
[Detailed current state in prospect's language]

OBJECTIVE STATE MAP:
[Vivid desired future in prospect's language]

GAP ARCHITECTURE:
Problem #1: [Surface → Real → Root Cause → Personal Impact]
Problem #2: [Surface → Real → Root Cause → Personal Impact]
Problem #3: [Surface → Real → Root Cause → Personal Impact]
[Up to 5 problems]

AMPLIFICATION SEQUENCE:
[Problem mirror + future projection + consequence stack]

BRIDGE TO SOLUTION:
[Self-persuasion questions that let them conclude they need your help]
```

## Industry-Specific Question Banks

Generate question frameworks customized to the prospect's industry. Reference the transcript's examples across: healthcare, chiropractic, real estate, insurance, financial services, solar, marketing agencies, roofing, health supplements, and B2B services.

## Quality Gate

- [ ] Did the prospect articulate AT LEAST 3 problems they didn't mention initially?
- [ ] Is every problem drilled to Layer 3 (personal impact)?
- [ ] Does the gap feel too painful to tolerate (not just "inconvenient")?
- [ ] Did the PROSPECT build the bridge to the solution, not you?
- [ ] Are all tonal annotations correct (concern tone for Layer 3)?
