# Copy Psychology Injector

> Take anything persuasive and make it psychologically devastating. This workflow takes existing copy and injects Jeremy Miner's psychological trigger architecture.

## When to Use

- After writing a landing page (inject triggers before publishing)
- After creating a sales email sequence (upgrade each email)
- After drafting LinkedIn posts or content (add self-persuasion hooks)
- After building a sales script (add tonal annotations + frames)
- When reviewing existing copy that's underperforming
- As a "second pass" after ANY content creation workflow

## Inputs Required

1. **The copy/content** to be upgraded
2. **The target audience** (who is reading/hearing this?)
3. **The desired action** (what should they do after consuming this?)
4. **Top 3 objections** the audience would have
5. **The identity shift** needed (who are they now → who do they become?)

## The 7-Layer Injection Process

### Layer 1: Frame Analysis

Read the copy and identify:
- What FRAME does the copy currently put the reader in?
- Is it a golden frame (high perceived value) or flea market frame?
- Does the opening calm the nervous system or trigger it?

**Upgrade:** If the copy opens with hype, urgency, or pressure → rewrite to calm the nervous system first. Replace "Limited time!" with confident, unhurried authority.

### Layer 2: Self-Persuasion Hook Injection

Find every place the copy TELLS the reader something and convert it to a QUESTION that makes them tell themselves.

**Before:** "Your current marketing isn't working."
**After:** "Are you 100% satisfied with the results your marketing is getting? Or is there a gap between where you are... and where you know you could be?"

**Rules:**
- Headlines: convert to self-persuasion questions
- Subheadings: add 100% framing or consequence awareness questions
- Body copy: replace statements with "What if..." and "Have you noticed..."

### Layer 3: Identity Frame Installation

Find the copy's core value proposition and wrap it in identity language.

**Before:** "Our program helps you grow your business."
**After:** "The entrepreneurs who grow fastest have one thing in common — they stopped trying to figure everything out alone. If you're the type who invests in the right solutions rather than grinding through problems... this was built for you."

**Injection points:**
- Lead paragraph: identity label ("If you're the type who...")
- CTA section: identity reinforcement ("People like you don't wait...")
- Testimonials: reframe as identity stories ("I used to be the type who...")

### Layer 4: Gap Architecture

Ensure the copy builds a visible gap between current state and objective state.

**Check for:**
- Does the copy clearly establish CURRENT STATE (pain, problems)?
- Does the copy clearly establish OBJECTIVE STATE (desired future)?
- Is the gap between them wide enough and painful enough?

**If the gap is weak:**
- Add problem awareness sections using the 3-Layer Drill
- Add consequence stacking (what happens if nothing changes)
- Add "in reality... which is more expensive?" comparison
- Add personal impact language ("What toll has this taken on your...")

### Layer 5: Objection Prevention Injection

For each of the top 3 objections, inject prevention frames:

| Objection Type | Injection |
|----------------|-----------|
| **Price** | Add intention frame: "This isn't about the money. It's about [result]. What's going to put you in the best position to get there?" |
| **Timing** | Add identity frame: "The people who get results don't wait for the 'right time.' They create it." |
| **Trust** | Add social proof + negative contrast: "Others hesitate. The ones who [achieved result] decided differently." |
| **Authority** | Add necessity frame: "You wouldn't wait to fix [urgent necessity]. This is the same thing." |
| **Comparison** | Add neutral doubt: "Other solutions are... fairly decent. But [differentiation]." |

### Layer 6: Tonal Calibration

Annotate the copy with tonal direction:

| Copy Section | Tone |
|-------------|------|
| Headline | 🟣 Playful or 🔵 Curious (interrupt pattern) |
| Problem section | 🟢 Concern (empathy, not attack) |
| Consequence section | 🟢 Concern + light 🔴 Challenge |
| Solution section | 🔵 Curious + calm authority |
| CTA | 🟢 Concern + 🔵 Curious (never 🔴 pressuring) |
| Testimonials | Authentic voice (no tonal manipulation) |

### Layer 7: Pacing Injection

Add pacing to the copy's visual rhythm:

- **Short paragraphs** after heavy emotional content (let it breathe)
- **Single-line paragraphs** for emphasis
- **Ellipses** for written verbal pacing: "What if you could... finally..."
- **Line breaks** between major sections
- **White space** as a strategic tool (not laziness)

## Output Format

Deliver:
1. **Injection Audit** — what the original copy was missing (organized by layer)
2. **Upgraded Copy** — the full rewritten version with all 7 layers applied
3. **Change Log** — every specific change made, with rationale
4. **Before/After Highlights** — 3-5 most impactful transformations
5. **Tonal Map** — the emotional journey the reader now takes

## Quality Gate

- [ ] Does the copy calm the nervous system in the first 3 seconds?
- [ ] Are there at least 3 self-persuasion questions replacing statements?
- [ ] Is identity language present in the opening AND the CTA?
- [ ] Is the gap between current state and objective state visceral?
- [ ] Are objection preventions embedded BEFORE the CTA?
- [ ] Does the overall tone feel like concern, not pressure?
- [ ] Would a reader feel they DISCOVERED a truth, not got SOLD?
