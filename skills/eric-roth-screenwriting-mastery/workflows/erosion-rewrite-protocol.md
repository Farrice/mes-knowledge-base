# Erosion Rewrite Protocol

Apply Eric Roth's signature technique: start from page one, read forward, and refine through natural accumulation — like water shaping stone. This is not rewriting. This is erosion.

## Pre-Flight

Read these files before executing:
1. `skills/eric-roth-screenwriting-mastery/genius.md` (principle #1 — Erosion > Revision)
2. `skills/eric-roth-screenwriting-mastery/references/genius-patterns.md` (Pattern 1)
3. `skills/eric-roth-screenwriting-mastery/references/hidden-knowledge.md` (HK-1)

## Input Required

- **Complete draft** — the script/manuscript/long-form piece to erode
- **Theme statement** — one sentence (if not yet articulated, do this first)
- **Known problems** (optional) — areas the writer already suspects need work

---

## Workflow

### Step 1: The Full Re-Entry

Read from page one. Do not skip ahead. Do not start where you "left off." The entire point is to re-enter the piece as a first-time reader.

As you read forward, mark each passage with one of three tags:

| Tag | Meaning | Action |
|-----|---------|--------|
| 🟢 **ALIVE** | This passage has energy, feels fresh, moves | Leave it. Maybe strengthen a word. |
| 🟡 **TIRED** | This passage works but feels flat or familiar | Shore it up — add specificity, sensory detail, or subtext |
| 🔴 **FALLING** | This passage is actively hurting the piece | Rebuild from scratch using the scene's purpose |

---

### Step 2: Shore Up the Tired

For each 🟡 passage, apply one or more of these erosion techniques:

- **Add sensory detail** (Pattern 7) — What does this scene feel like, sound like, smell like?
- **Shift time of day** (HK-8) — Would dusk create more melancholy? Dawn more hope?
- **Change the weather** (Pattern 8) — Would rain unlock new character behavior?
- **Deepen subtext** (Pattern 5) — Is anyone saying exactly what they mean? Displace it.
- **Add wallet detail** (Pattern 4) — Is the character behaving like someone with a specific psychology, or a generic person?
- **Tighten** — Can any words be removed without losing meaning?

---

### Step 3: Rebuild the Falling

For each 🔴 passage:

1. **State the scene's purpose in one sentence** — Why does this scene exist?
2. **Verify theme connection** — Does this scene relate to the compass? If not, cut or reimagine.
3. **Rebuild from purpose** — Write the scene fresh, using the purpose as the only constraint.
4. **Apply visual prose** — Don't just fix the dialogue; rebuild the sensory environment.

---

### Step 4: Continuity Integration

After all erosion passes, read straight through again from page one. This time, check for:

- **Tonal consistency** — Does the voice feel unified, or do eroded sections feel "patched in"?
- **Character continuity** — Do character voices remain distinct throughout?
- **Theme accumulation** — Does the theme grow richer as the reader progresses, or does it peak too early?
- **Pacing** — Are emotional peaks getting more real estate than setup? (Pattern 11: Slow parts fast, fast parts slow)

---

### Step 5: The Physical Checkpoint (HK-1 — adapted)

Roth's DOS program forces a physical checkpoint every 40 pages. Adapt this discipline:

- After every 30-40 pages of erosion, **stop and summarize** what the piece accomplishes so far
- Ask: "If the reader stopped here, would they know what this is about?"
- Ask: "Is the erosion adding layers, or just moving words around?"

## Output Format

```
EROSION PASS SUMMARY:

SECTION [pages/chapters]:
- 🟢 ALIVE passages: [count]
- 🟡 TIRED passages (shored up): [count] — techniques applied: [list]
- 🔴 FALLING passages (rebuilt): [count]

KEY CHANGES:
1. [Specific change and rationale]
2. [Specific change and rationale]
...

THEME ACCUMULATION CHECK:
- Theme connection by act: [strong/adequate/weak for each act]

CONTINUITY NOTES:
- [Any tonal, character, or pacing issues identified and addressed]

ERODED DRAFT:
[The full revised text with all erosion applied]
```

## Quality Gate

- [ ] Every page has been re-entered from page one (no skipping)
- [ ] All 🔴 passages have been rebuilt (not just patched)
- [ ] Visual prose added where scene descriptions were flat
- [ ] Theme accumulates — each act builds on the last
- [ ] Character voices remain distinct throughout
- [ ] The piece feels *more alive*, not just *more correct*
