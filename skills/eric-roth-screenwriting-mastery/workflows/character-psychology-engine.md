# Character Psychology Engine

Build psychologically complete characters using Eric Roth's "wallet" method — not cardboard heroes, but fanciful, specific, psychologically true human beings that audiences think are "somebody worth caring about."

## Pre-Flight

Read these files before executing:
1. `skills/eric-roth-screenwriting-mastery/genius.md`
2. `skills/eric-roth-screenwriting-mastery/references/genius-patterns.md` (Patterns 4, 6)
3. `skills/eric-roth-screenwriting-mastery/references/hidden-knowledge.md` (HK-9, HK-10)

## Input Required

- **Character name** (or placeholder)
- **Story context** — genre, setting, era, theme
- **Role** — protagonist, antagonist, supporting, mentor, etc.
- **Initial sketch** (if any) — whatever the writer already knows about this character

---

## Workflow

### Step 1: The Fanciful-Real Calibration (Pattern 6)

Before building the wallet, answer:

> This character is *fanciful* because ______________.
> This character is *real* because ______________.

The fanciful element is what makes them compelling (Forrest's innocence, Button's reverse aging, Tobacco executive's moral crisis). The real element is what makes audiences cry (a mother's love, a daughter's judgment, a man's isolation).

**Both must be present.** A character that is only fanciful is a cartoon. A character that is only real is a documentary subject.

---

### Step 2: The Full Wallet (Pattern 4)

Build the complete psychology wallet:

#### Identity Core
| Element | Details |
|---------|---------|
| **Full name** | |
| **Age** (at story start and end) | |
| **Physical description** | Not just appearance — how they *carry* themselves |
| **Occupation / Role** | What they do, and what they *wish* they did |

#### Emotional Architecture
| Element | Details |
|---------|---------|
| **What they love** (3 specific things) | Not categories — specific: "the smell of gasoline," not "cars" |
| **What they fear** (2 specific things) | The surface fear AND the deeper fear beneath it |
| **Neuroses** | Specific behavioral quirks (checks locks twice, can't eat in front of people) |
| **Anger triggers** | What makes them disproportionately angry — reveals wounds |
| **Giddiness** | What makes them childishly, irrationally happy |
| **Secret shame** | The thing they'd never tell anyone |
| **Defining loss** | The event that shaped their emotional landscape |

#### Voice Fingerprint
| Element | Details |
|---------|---------|
| **Vocabulary level** | Education and background expressed through word choice |
| **Speech rhythm** | Short bursts? Long flowing sentences? Interrupts? Pauses? |
| **Verbal tics** | Repeated phrases, filler words, rhetorical habits |
| **Subtext default** | Do they say what they mean, or always deflect? |
| **Silence behavior** | What they do when they don't speak — fidget, stare, leave? |

#### The Literal Wallet
What's physically in their back pocket, purse, or phone:
- A crumpled phone number from ____________
- A picture of ____________
- A saying they find interesting: "____________"
- An object that has no logical reason to be there

#### Theme Connection
- How does this character **embody** the theme?
- How does this character **challenge** the theme?
- What would this character say if asked directly about the theme?

---

### Step 3: The Transformation Arc (Pattern 5 from Albom — adapted)

> This character begins as [STATE A] and ends as [STATE B].

Map 3-5 beats between A and B:
1. **Inciting shift** — first crack in their existing identity
2. **False solution** — they try to change the wrong thing
3. **Mirror moment** — they see themselves clearly for the first time
4. **Commitment** — they choose the painful path toward B
5. **Integration** — B becomes their new operating system

---

### Step 4: Voice Isolation Test

Write three short passages to test voice distinction:

1. **Ordering food at a restaurant** (reveals social personality)
2. **Describing a person they love** (reveals emotional depth)
3. **Responding to unexpected bad news** (reveals defensive architecture)

**Test**: Could someone identify this character from these passages without a name tag?

---

### Step 5: The Roth Paradox Check

Final validation:
- [ ] Is this character fanciful enough to be compelling?
- [ ] Is this character real enough to cry for?
- [ ] Does this character have at least 2 things they love?
- [ ] Would an actor be excited to play this role?
- [ ] Does this character's voice sound *different* from every other character?
- [ ] Does the literal wallet contain at least one surprising item?
