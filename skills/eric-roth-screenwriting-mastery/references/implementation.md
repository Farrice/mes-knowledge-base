# Implementation Guide

How to deploy Eric Roth Screenwriting Mastery — standalone and stacked with other skills.

---

## Standalone Deployment

### For Screenwriters
1. Load `genius.md` for voice and principles
2. Start with **Screenplay Architect** workflow for structure
3. Use **Character Psychology Engine** for each major character
4. Apply **Theme Compass Navigator** to ensure thematic coherence
5. Run **Erosion Rewrite Protocol** on completed drafts
6. Polish with **Visual Prose Cinematographer** for scene descriptions

### For Content Creators / Storytellers
1. Load `genius.md`
2. Use **Character Psychology Engine** to build personas
3. Apply **Subtext Alchemist** to elevate dialogue in scripts/copy
4. Use **Memorable Moment Forge** for hooks and quotable moments
5. Apply **Opening Scene Architect** principles to content openings

### For Adaptation Projects
1. Load `genius.md`
2. Start with **Adaptation Transmuter** for source analysis
3. Build characters via **Character Psychology Engine**
4. Structure via **Screenplay Architect** or **Time Architecture Engine**
5. Erosion pass when draft is complete

## Anti-Patterns

| ❌ Don't | ✅ Do Instead |
|----------|--------------|
| Copy Roth's exact dialogue patterns | Absorb the *off-center* principle and apply to your own work |
| Write 200-page scripts thinking "more prose = better" | Use visual prose selectively — density in key moments, efficiency elsewhere |
| Force time-spanning structure into every project | Use Time Architecture Engine only when time serves the theme |
| Treat the erosion protocol as "start over" | Erosion refines — it never discards |
| Skip the character wallet for minor characters | Even supporting characters need 2-3 wallet items for voice distinction |
