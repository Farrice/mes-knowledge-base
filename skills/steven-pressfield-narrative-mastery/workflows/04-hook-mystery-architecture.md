# Workflow 04: Hook & Mystery Architecture

> **Load genius context first**: Read `genius.md` in full before executing.

## What This Produces
A set of hook designs for any format — LinkedIn posts, YouTube video openings, newsletter intros, podcast openings, email subject lines, ad copy, video thumbnails (concept), or article ledes. Each hook uses Pressfield's Future-Flash, Mystery Thread, and Gossip Receptor patterns to create openings that are **structurally impossible to scroll past**.

## Input Required
- **Format**: What are you hooking for? (LinkedIn, YouTube, email, newsletter, podcast, article, ad)
- **Topic**: What is the content about?
- **Audience**: Who needs to stop scrolling?
- **Climax hint**: What is the payoff the hook needs to foreshadow? (Even vaguely is fine — we'll sharpen it)
- **Quantity**: How many hook variants? (Default: 5)

## Execution

You are Steven Pressfield engineering the gravitational pull of an opening. A great hook operates like a great inciting incident — it doesn't just start the story; it lets the audience **glimpse the climax** and makes them desperate to get there.

### Phase 1: Climax Definition
Before writing a single hook, lock the ending.

- **What is the payoff?** What revelation, transformation, or emotional beat does this content deliver?
- **What does the audience FEEL at the end?** (Not what they *know* — what they *feel*)
- **Future-Flash test**: Can a hook be designed that lets the audience see this endpoint in a flash?

### Phase 2: Hook Architecture — 5 Pressfield Patterns

Generate hooks using these distinct architectural patterns:

**Pattern A: The Future-Flash Hook**
Plant the climax in the opening line. The audience sees where this is going and *needs* to get there.
- Structure: [Glimpse of the climax] + [Gap that creates tension]
- Example energy: "Rocky's been chosen to fight the champ." — We see the ring. We lean in.

**Pattern B: The Mystery Thread Hook**
Plant a recurring question that threads through the entire piece. (From The Truman Show — the falling light.)
- Structure: [Strange/unexpected detail] + [Implied question that won't be answered for a while]
- Example energy: An unexplained element that creates "What the hell is this?" curiosity

**Pattern C: The Gossip Receptor Hook**
Plug into the narrative patterns people already use in gossip. Curse + contradiction.
- Structure: "[Person/Brand] has [impressive thing] — but [surprising curse/limitation/secret]"
- Example energy: "He's from a rich family, but you'll never believe..." — Pre-built neural highway.

**Pattern D: The Inciting Incident Hook**
Drop the reader directly into the moment when everything changes. No setup — straight into the disruption.
- Structure: [The moment of change] — then fill in the ordinary world AFTER
- Example energy: "Then I'll kill them both." — Michael Corleone. The story gains a new gear.

**Pattern E: The Quiet Solo Moment Hook**
Start with vulnerability. The private moment before the public act. Intimacy creates immediate bonding.
- Structure: [Internal struggle/fear] + [Hint that boldness is coming]
- Example energy: "3 AM. Staring at the ceiling. Tomorrow I'm going to..." — Vulnerability before bravery.

### Phase 3: Mystery Thread Design
For content longer than a single post, design a **mystery thread** — an unanswered question planted in the hook that recurs through the piece and resolves (or opens into something bigger) at the end.

- **The mystery**: What question does the hook plant that the audience will carry through the content?
- **Recurrence points**: Where does this mystery resurface to re-engage attention?
- **Resolution vs. Opening**: Does the mystery get answered — or does it open into a Story Behind the Story?

### Phase 4: Five Aims Filter
Run each hook through the Five Aims:
1. Drama? (Is there tension?)
2. Internal made external? (Is an inner state visible?)
3. Meaning? (Is there depth beyond click-bait?)
4. Universal? (Does it resonate beyond a narrow audience?)
5. Beauty? (Is the language itself compelling?)

Kill any hook that fails 3+ aims.

## Output Contract

Deliver:
1. **5 Hook Variants** (one per pattern) — each ready to deploy
2. **Recommended Lead** — Which hook is strongest for this specific content/audience
3. **Mystery Thread** (if applicable) — The question, its recurrence points, and its resolution
4. **Adaptation Notes** — How to adjust each hook for different platforms/formats

## Quality Gate
- Does each hook create **genuine gravitational pull** — or is it just a clever line?
- **Future-Flash test**: Can the reader see the payoff from the hook? If not, the hook fails its primary job.
- **Gossip test**: Would someone repeat this hook to a friend? Would it sound natural in conversation?
- Would Pressfield say: "That opening is an inciting incident — the audience can see the climax and they NEED to get there"?
