# Mechanism Discovery Engine

Systematically brainstorm unique mechanisms for any product or offer using Luke Iha's Universal Mechanism Matrix + 6 insight vectors. Produces a ranked shortlist filtered through the SIN framework.

---

## Inputs Required

1. **Product/Offer** — What are we selling? What does it actually DO?
2. **Core Problem** — What pain does the audience experience?
3. **Product Truths** — What is genuinely new, different, or unique about this product?
4. **Market Context** — How sophisticated is this market? How many competing claims exist?
5. **Audience Intelligence** — What does the audience already suspect or believe?

## Phase 2: Universal Mechanism Matrix

Run the core problem through the 3×4 matrix:

### Matrix Template

|  | Structure | Function | Element |
|--|-----------|----------|---------| 
| **Too Much** | "[body/system] has too many [structural thing]" | "[body] over-produces [process]" | "There's too much [substance] in your [system]" |
| **Too Little** | "[structure] is missing [component]" | "[body] can't produce enough [process]" | "You're deficient in [element]" |
| **Out of Balance** | "[structure] is misaligned/displaced" | "[process A] is fighting [process B]" | "[Element A] is blocking [Element B]" |
| **Dysfunctional** | "[structure] is damaged by [external cause]" | "[process] has been hijacked by [thing]" | "[External element] has corrupted your [system]" |

### Instructions:
1. Fill each cell with a specific mechanism candidate related to the product's problem
2. Don't force cells that don't apply — aim for ≥ 8/12
3. Each candidate should be ONE sentence
4. Frame candidates through the UMP/UMS lens chosen in Phase 1

---

## Phase 3: Insight Vector Mining

For each mechanism candidate that shows promise, mine deeper using the 6 insight vectors:

1. **Scientific Studies** — PubMed, Google Scholar. Surprising correlations, newly discovered pathways
2. **Ancestral Wisdom** — Traditional remedies, Ayurvedic/TCM. "Ancient + recently validated" = powerful
3. **Practitioner Observations** — What clinicians notice that's not in textbooks
4. **Patient Case Studies** — Outlier results. What happened differently for them?
5. **Adjacent Fields** — Mechanisms from one field applied to another
6. **Failed Experiments** — What was supposed to fail but didn't? Serendipity mechanisms

### For each promising candidate:
- Which insight vectors support it?
- What would a 30-second "discovery story" sound like?
- Does it tap into something the audience already suspects?

---

## Phase 4: SIN Filter

Score each surviving candidate:

| Candidate | Simple (1-10) | Intuitive (1-10) | New (1-10) | Total | Pass? |
|-----------|---------------|-------------------|------------|-------|-------|
| [name] | | | | /30 | ≥21? |

- **Simple**: Can you explain it to a 12-year-old in one sentence?
- **Intuitive**: Does it "feel right" before proof? Does the audience already suspect this?
- **New**: Has the audience heard this before? Does it feel like a discovery?

**Kill rule**: < 15 → abandon. 15-20 → refine. ≥ 21 → proceed.

---

## Phase 5: Characterization (Naming)

For each mechanism that passes SIN, generate 5+ name candidates using the naming formulas:

- [Vivid Adjective] + [Body/Thing]: "Toxic Calcium," "Silent Inflammation"
- [Thing] + [Action]: "Cortisol Switch," "Fatigue Loop"
- The [Hidden/Secret/Forgotten] + [Effect]: "The Hidden Hunger Signal"
- [Surprising Noun] + [Problem Area]: "Sugar Brain," "Plastic Gut"

### Name Quality Test:
- ☐ 2-3 words maximum
- ☐ Visual — creates an image
- ☐ Emotional — triggers a feeling
- ☐ Enemy-coded — sounds like something to eliminate
- ☐ Dinner table test — would a non-expert remember it?

## Quality Gate

Before delivering:
- ☐ At least 8 matrix cells filled
- ☐ At least 3 candidates passed initial SIN filter
- ☐ Winning mechanism has SIN ≥ 21
- ☐ Name passes all 5 characterization tests
- ☐ Can articulate why UMP/UMS was chosen in one sentence
- ☐ Mechanism taps into a pre-existing audience suspicion (not invented from nothing)
