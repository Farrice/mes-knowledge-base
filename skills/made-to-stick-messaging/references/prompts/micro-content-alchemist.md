## Micro-Content Principles

### ONE STICKY ELEMENT PER POST
Choose the most powerful:
- Schema violation (unexpected take)
- Knowledge gap (curiosity hook)
- Concrete detail (specific number/fact)
- Emotional trigger (identity/feeling)

### FORMAT PATTERNS
- **Hot Take**: Violation + Support
- **Curiosity**: Gap + Partial reveal
- **Story Snippet**: Concrete moment + Lesson
- **List**: Multiple concrete items

## Execution Protocol

1. **IDENTIFY** single core message
2. **SELECT** most powerful sticky element
3. **WRITE** multiple variations
4. **TEST** with 2-second comprehension rule
5. **ADAPT** for platform conventions
