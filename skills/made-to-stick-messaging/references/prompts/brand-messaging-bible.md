# Brand Messaging Bible

> Comprehensive document capturing all brand messaging using sticky principles.

## Role & Activation

You are deploying Heath Brothers methodology for brand codification. A messaging bible ensures all communication is sticky and consistent.

Core insight: Every brand touchpoint should reinforce the same Commander's Intent using consistent sticky elements.

## Input Required

- **[BRAND]**: What brand is this for?
- **[AUDIENCE]**: Who does this brand serve?
- **[DIFFERENTIATION]**: What makes this brand different?
- **[EXISTING_MATERIALS]**: Current messaging, content, positioning?

## Messaging Bible Sections

### COMMANDER'S INTENT
The single thing all messaging must convey

### BRAND STORY
The narrative that defines who we are

### AUDIENCE PROFILES
Deep understanding using ICP excavation

### MESSAGING PILLARS
3-5 key themes with sticky translations

### VOCABULARY
Words we use / words we avoid

### PROOF BANK
Stories, stats, testimonials organized

### TONE & VOICE
How we sound (with examples)

### APPLICATION GUIDES
How to apply for: social, email, ads, sales

## Execution Protocol

1. **EXTRACT** Commander's Intent
2. **CRAFT** brand story
3. **DEFINE** audience deeply
4. **CREATE** messaging pillars
5. **BUILD** proof bank
6. **DOCUMENT** voice and vocabulary
7. **CREATE** application guides

## Output Deliverable

Complete messaging bible:
- All sections above
- Quick-reference one-pager
- Examples for each section
- Template library
