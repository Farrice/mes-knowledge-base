# Micro Lead Generator

Mass micro lead production engine. Generates one micro lead of each of the 7 proven types for any offer, scores them, and recommends the top 3 for testing. Each micro lead is testable independently as a social post, email opener, or ad hook.

## PHASE 2: INPUT REQUIREMENTS

Collect from user:
- **Product/Offer**: What's being sold?
- **Target Audience**: Who is this for?
- **Awareness Level**: Unaware / Problem-aware / Solution-aware / Product-aware / Most-aware
- **Core Problem**: The main pain
- **Surprising Fact/Insight**: What's the most counterintuitive thing about this product/problem?
- **Key Credential**: Best proof point (person, data, institution)

## PHASE 3: GENERATE ALL 7 TYPES

For each type, write a 50-200 word micro lead:

### Type 1: Contrarian Claim
Bold statement contradicting common belief. Open with the claim, then immediately create stakes.

### Type 2: Insider Confession
Someone who "shouldn't" reveal truth, reveals it. Create the sense of forbidden knowledge.

### Type 3: Paradoxical Question
A question whose answer seems logically impossible. The answer can only be found by continuing.

### Type 4: Look Closer
Draw attention to something hiding in plain sight. Make the familiar suddenly strange and significant.

### Type 5: Surprise Connection
Link two seemingly unrelated things. The connection itself is the hook — resolution requires reading on.

### Type 6: Invisible Difference
Present two things that appear identical but have drastically different outcomes. Create anxiety about which side the reader is on.

### Type 7: Quiz Opening
Engage through self-categorization. One question that reveals a hidden truth about the reader.

## PHASE 4: SCORING

Score each micro lead on 4 dimensions (1-10 each):

| Dimension | Question |
|-----------|----------|
| **Curiosity** | Does this create an open loop that MUST be resolved? |
| **Relevance** | Does the target audience FEEL this in their bones? |
| **Specificity** | Does it contain concrete, believable details? |
| **Momentum** | Does the reader NEED to read the next sentence? |
