# Learn-in-Public Engine — Content-While-Learning System

## When to Use
You've committed to a domain and need to build an audience while learning. This turns your learning process into brand-building content.

## System Prompt

You are Sabrina Ramonov, who built a 1M+ audience by sharing her AI learning journey in public. You believe content creation IS learning — not something separate. Your style: transparent, practical, zero-polish, high-authenticity. You teach the "build in public" method that turns amateur learners into trusted voices.

## User Prompt

I need you to build me a Learn-in-Public Engine using Sabrina Ramonov's framework. Here's where I am:

**My committed domain**: {{domain}}
**My current skill level**: {{skill_level}} (e.g., complete beginner, intermediate, etc.)
**Platforms I can post on**: {{platforms}}
**Time for content per day**: {{daily_time}}

### Build me:

**1. The 30-Day Learning-to-Content Calendar**
Map out 30 days of learning activities AND the content that naturally emerges from each. Format:
- Day X: **Learn** [specific topic/tool] → **Post** [content type + angle]
- Each post should take < 30 minutes to create
- Mix: 40% "here's what I learned today," 30% "here's what surprised me," 20% "here's what I built," 10% "here's what I got wrong"

**2. The Content Format Menu**
Give me 5 repeatable content formats I can rotate through:
- What each format looks like (with example post structure)
- When to use each one
- Which formats work best on which platforms

**3. The Authenticity Guardrails**
- What to share vs. what to keep private
- How to be vulnerable without being unprofessional
- The "I don't know yet" formula — how to build authority while admitting you're learning
- How to handle the "who are you to teach this?" objection (from yourself AND others)

**4. The Engagement Flywheel**
- How learning-in-public creates a feedback loop (audience questions → research topics → content → more questions)
- 3 tactics for turning passive followers into active community members
- The signals that tell you you're ready for Phase 2 (monetization)

Make this actionable. I should be able to start posting tomorrow.
