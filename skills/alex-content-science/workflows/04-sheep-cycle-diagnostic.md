# Sheep Cycle Diagnostic

> Load `skills/alex-content-science/genius.md` before executing.

## Inputs Required
- **Content audit** — links to or descriptions of last 10-20 pieces of published content
- **Performance data** (optional) — which performed best/worst?
- **Honest self-assessment** — "what percentage of my content is inspired by specific competitors?"

## The Process

### Step 1 — The Cycle Mapping

Map the creator's recent content history onto the Sheep Cycle:

```
COPY competitor → OK views → COPY another → OK views → TRY original → LOW views → PANIC → COPY more → STUCK
```

For each of the last 10-20 pieces, tag:
- **[C]** = Clearly inspired by/copied from specific competitor content
- **[S]** = Similar format/topic to what everyone in niche is doing
- **[O]** = Genuinely original — no obvious source
- **[P]** = Panic post (created after a low-performing original piece)

Count the ratio: [C + S] vs [O].

If C + S > 60% → **Active Sheep Cycle confirmed.**

### Step 2 — The Interchangeability Test

Take the 5 best-performing pieces and run the cover test:
- Cover the account name/profile picture
- Could this content belong to ANY of 5 competitors?
- Score each: 1 (completely unique) to 5 (completely interchangeable)

Average score > 3 → **Content is brown cow territory.**

### Step 3 — Audience Quality Audit

Analyze WHO the content is attracting:
- Are followers engaging with EVERY post, or only the ones that match the copied format?
- When non-copied content is posted, do engagement numbers collapse? (This is the key signal.)
- Are comments about YOU or about the TOPIC? (Topic comments without personal connection = trend followers, not fans.)
- DMs/replies: do people say "I love how YOU explain things" or just "great tip"?

### Step 4 — Root Cause Diagnosis

Common Sheep Cycle root causes:

| Root Cause | Signal | Treatment |
|-----------|--------|-----------|
| **Fear of low views** | Original posts = anxiety; copied posts = comfort | Redefine success metrics (brand equity > views) |
| **No unique perspective** | Can't articulate what makes you different | Run the "What Do I Refuse?" exercise |
| **Research-as-procrastination** | "Studying competitors" is actually avoiding creation | Limit research to 1 hr/week max; create 5x more |
| **Format dependency** | Only knows how to make one type of content | Cross-niche format hacking sprint |
| **Audience-trained on copies** | Followers expect trend content; originals confuse them | Gradual transition (70/30 → 50/50 → 30/70) |

### Step 5 — The Escape Plan

Three phases to break the cycle:

**Phase 1 — Awareness (Week 1-2)**
- Tag all content as [C], [S], [O], [P] going forward
- Run Detail Stack breakdown on 3 competitor pieces you've copied
- Extract principles from those breakdowns (don't copy them again)
- Document: "What do I refuse to do?" — define brand through exclusion

**Phase 2 — Transition (Week 3-6)**
- Content ratio: 50% principle-based originals / 50% niche-standard
- For every niche-standard post, pair it with one original
- Run the Purple Cow Test on everything before publishing
- Track: which originals perform? (Even low numbers = right audience)

**Phase 3 — Independence (Week 7+)**
- Content ratio: 80% originals / 20% format experiments
- All "inspiration" runs through the Detail Stack first (no surface copying)
- Monthly cross-niche format scan for fresh structural inspiration
- Build brand system: consistent environment, wardrobe, audio, pacing

## Output Format

### Sheep Cycle Diagnostic Report

**Subject**: [creator name/brand]

**Cycle Status**: [Active / Early-Stage / Not Present]

**Content Ratio**: [C]% Copied | [S]% Similar | [O]% Original | [P]% Panic

**Interchangeability Score**: [1-5 average]

**Audience Quality**: [Trend-followers / Mixed / True fans]

**Root Cause**: [primary diagnosis]

**Escape Plan**: [3-phase plan customized to diagnosis]

**30-Day Check-In Metrics**:
- Original content ratio target: [X]%
- Purple Cow score target: [X]
- New formats attempted: [X]

- [ ] All 10-20 content pieces tagged with honest [C]/[S]/[O]/[P] labels

- [ ] Root cause identified with supporting evidence (not assumption)
- [ ] Escape plan is specific and phased (not "just be more original")
