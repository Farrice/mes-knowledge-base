# Principle Extraction Engine

> Load `skills/alex-content-science/genius.md` before executing.

## Inputs Required
- **Detail Stack Breakdown(s)** — one or more completed breakdowns from Workflow 01
- **Target niche** (optional — what niche will principles be applied to?)

## The Process

### Step 1 — Surface the Mechanisms

For each Detail Stack Breakdown, identify the MECHANISMS that drove success:

| Content Element | What They Did | The Mechanism |
|----------------|---------------|---------------|
| Hook | [specific observation] | [why it worked psychologically] |
| Structure | [specific observation] | [what structural pattern created engagement?] |
| Visual Design | [specific observation] | [what design principle was leveraged?] |
| Audio | [specific observation] | [what auditory principle?] |
| Identity Signal | [specific observation] | [what trust/authority mechanism?] |
| Emotional Arc | [specific observation] | [what emotional journey was engineered?] |

### Step 2 — The One-Sentence Principle Test

For each mechanism, extract a one-sentence principle. Rules:

1. **ZERO references** to the specific video, creator, topic, or platform
2. **Describes the WHY**, not the WHAT
3. **Applies to 10+ completely different niches** without modification
4. **Uses "they" or generic subject**, never the specific person's name

**Test each principle**: Read it aloud. If someone could guess the original content from the principle alone — it's still a description, not a principle.

**Examples of BAD extractions:**
- ❌ "He made a Kai Cenat video about public speaking" (that's description)
- ❌ "Using red socks creates visual interest" (too specific/tactical)
- ❌ "Videos about vocabulary should use fashion analogies" (topic-locked)

**Examples of GOOD extractions:**
- ✅ "A single contrasting visual detail in an otherwise muted frame keeps eyes inside the frame while signaling personality"
- ✅ "Framing yourself as the coach to a recognized authority figure repositions you as an expert-of-experts through sentence structure alone"
- ✅ "Mapping a high-value skill people avoid onto a high-interest domain they already love transforms homework into entertainment"

### Step 3 — Principle Categorization

Sort extracted principles into deployment categories:

| Category | Principle | Deployment Scope |
|----------|----------|-----------------|
| **Visual** | [principles about visual decisions] | Thumbnail, video, set design |
| **Structural** | [principles about content architecture] | Scripts, hooks, pacing |
| **Psychological** | [principles about trust/authority/status] | Positioning, framing |
| **Emotional** | [principles about feelings/triggers] | Hooks, narratives, CTAs |
| **Sonic** | [principles about audio decisions] | Music, voice, silence |
| **Identity** | [principles about brand/persona signals] | Wardrobe, environment, consistency |

### Step 4 — Transferability Scoring

Score each principle 1-5 on:
- **Universality** (1 = niche-specific, 5 = works in any domain)
- **Actionability** (1 = abstract insight, 5 = immediately deployable)
- **Originality** (1 = common knowledge, 5 = nobody teaches this)

Prioritize principles scoring 4+ in all three dimensions.

### Step 5 — Principle Stacking

Identify which principles compound when combined:
- "Pattern interrupt through silence" + "Status framing via sentence structure" = calm authority opening that repositions you as the expert
- "Environment selling" + "Identity wardrobe signal" + "Set dressing as status" = complete visual trust system

Document principle stacks that create compound effects.

## Output Format

### Principle Extraction Report

**Source**: [number of breakdowns analyzed]

**Principles Extracted**: [total number]

**Top 5 Priority Principles** (highest transferability scores):

1. **[Principle Name]**: "[one-sentence principle]"
   - Category: [Visual/Structural/Psychological/Emotional/Sonic/Identity]
   - Scores: Universality [X] | Actionability [X] | Originality [X]
   - Application hint: [one-sentence deployment suggestion]

[Repeat for top 5]

**Principle Stacks** (compound combinations):
- Stack 1: [Principle A + B] → [compound effect]
- Stack 2: [Principle C + D] → [compound effect]

**Ready for Remix**: These principles are now ready to feed into the Content Remix Lab (Workflow 03).

## Quality Gate
- [ ] Every principle passes the One-Sentence Principle Test (zero content references)
- [ ] At least 3 principles score 4+ on all three transferability dimensions
- [ ] At least 1 principle stack identified
- [ ] Principles cover 3+ categories (not all visual or all structural)
