# Content Remix Lab

> Load `skills/alex-content-science/genius.md` before executing.

## Inputs Required
- **Extracted principles** from Workflow 02 (at least 2)
- **Your niche/topic** — what domain is the content for?
- **Your audience** — who are you creating for?
- **Platform** — where will this be published? (YouTube, LinkedIn, Instagram, etc.)
- **Your brand identity** — brief description of your visual/verbal brand

## The Process

### Step 1 — Principle-to-Niche Translation

For each extracted principle, generate 3 specific applications in YOUR niche:

**Principle**: "[the one-sentence principle]"

| Application | How It Works in Your Niche | Format |
|------------|--------------------------|--------|
| Application A | [specific execution] | [short-form/long-form/carousel/etc.] |
| Application B | [specific execution] | [format] |
| Application C | [specific execution] | [format] |

### Step 2 — Cross-Pollination

Take the best application from each principle and mix them:
- Principle 1 application + Principle 2 application = Content Concept 1
- Principle 2 application + Principle 3 application = Content Concept 2
- Etc.

The goal is STACKING — multiple principles working together in a single piece of content, just like Alex demonstrates with the Detail Stack.

### Step 3 — The Originality Check

For each content concept, run:
1. **Purple Cow Test**: Could this be confused with any competitor's content? If yes → rework.
2. **Source Trace Test**: Could anyone trace this back to the original content you analyzed? If yes → the principle wasn't extracted deeply enough.
3. **Your Voice Test**: Does this sound like YOU, not the original creator? If not → filter through your brand identity.
4. **"What Am I Refusing?"** Test**: What conventional approach are you deliberately NOT taking? Document this — it's where your originality lives.

### Step 4 — Concept Development

For each passing concept, develop into a content brief:

**Content Concept [#]: [Working Title]**

- **Principle Stack**: [which extracted principles power this?]
- **Platform**: [where it publishes]
- **Format**: [specific format — talking head, carousel, thread, etc.]
- **Topic**: [what it's about]
- **Hook**: [opening line or visual — the stop signal]
- **Analogy** (if applicable): [boring topic mapped onto what high-interest domain?]
- **Invisible Decision Plan**: [3-5 deliberate Detail Stack decisions you're making]
  - Environment: [what will your setting signal?]
  - Audio: [music, silence, voice tone?]
  - Visual identity: [wardrobe, colors, objects?]
  - Pacing: [fast/slow, cut frequency?]
  - What you're REFUSING: [what convention are you breaking?]
- **Emotional Journey**: [how should the viewer feel → feel → feel by the end?]
- **One-Sentence Promise**: [what does the audience get from this?]

### Step 5 — Priority Ranking

Rank all concepts by:
1. **Originality** — how far is this from what exists?
2. **Achievability** — can you produce this with current resources?
3. **Impact** — how much will this stand out in the feed?
4. **Brand alignment** — does this build YOUR specific brand?

Pick top 3-5 for production.

## Output Format

### Content Remix Lab Output

**Principles Applied**: [list]
**Niche**: [your niche]
**Platform**: [platform]

**Top [3-5] Original Content Concepts**:

[Full concept brief for each as detailed in Step 4]

**Production Priority Order**: [1, 2, 3...]

**Next Steps**: [specific actions to produce concept #1]

## Quality Gate
- [ ] All concepts pass the Purple Cow Test (not interchangeable with competitors)
- [ ] All concepts pass the Source Trace Test (can't trace back to original)
- [ ] All concepts include 3+ deliberate Detail Stack decisions
- [ ] At least one concept uses the "What Am I Refusing?" filter
- [ ] Each concept has a clear emotional journey defined
