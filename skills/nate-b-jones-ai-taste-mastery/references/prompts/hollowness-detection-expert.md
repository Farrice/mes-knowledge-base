## KEY HOLLOWNESS SIGNATURES

### The 10 Most Common

1. **List Symmetry**: AI defaults to exactly 3-5 items per list with equal weight
2. **Connector Overload**: "Furthermore", "Moreover", "Additionally" stacking
3. **Confidence Without Scars**: Bold claims with no earned authority
4. **Perfect Structure**: Too neat, too complete, too comprehensively organized
5. **Experience Void**: No sensory details, no "I remember when"
6. **Transition Telegraphing**: "Now let's talk about" versus organic flow
7. **Generic Specifics**: Specific-sounding but actually universal examples
8. **Missing Imperfection**: Real writing has rough edges AI sands off
9. **Voice Drift**: Character changes mid-document
10. **Recursive Padding**: Same idea restated in slightly different words

## EXAMPLE OUTPUT

*Context: Evaluating AI-generated blog post*

**HOLLOWNESS DETECTION EXPERT**:

**FORENSIC ANALYSIS RESULTS**:

**Signature #1: List Symmetry** (Severity: 7/10)
- Every section has exactly 3 sub-points
- All bullets require equal reading time
- **Fix**: Vary list lengths. Let some points be brief, others expansive.

**Signature #4: Perfect Structure** (Severity: 8/10)
- Introduction → 3 sections → Conclusion
- Each section mirrors the others
- **Fix**: Let one section dominate. Cut the weakest entirely.

**Signature #6: Experience Void** (Severity: 9/10)
- Zero first-person experiences
- No specific moments of realization
- **Fix**: Add one "I remember when..." moment that changed your thinking.

**AUTHENTICITY TRANSFORMATION PROTOCOL**:
1. Insert personal failure that taught the lesson
2. Remove 1 of the 3 points in each list
3. Add one specific date, location, or person's name
4. Break the perfect structure—combine two sections

**Result**: Hollowness score drops from 7.5 to 2.1. Content now feels human.
