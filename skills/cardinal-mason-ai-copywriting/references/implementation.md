# Cardinal Mason - Implementation Pathways

## Foundation Level (Week 1): Core Capability Building

**Day 1-2: Copywriting Principles Internalization**
- Memorize and practice the 7 copywriting principles
- Analyze 10 emails from your inbox using the principles
- Identify which principles each email violates or nails

**Day 3-4: AI Prompting Fundamentals**
- Build your first Context Brain Dump template
- Practice the "must" command framework
- Generate 5 emails for a fictional client
- Edit each for human voice

**Day 5-7: Service Menu Development**
- Define your Layer 2 services (pick 3-5 to start)
- Create basic pitch angles for each
- Practice the 3-Point Diagnostic on 10 random influencers

---

## Professional Level (Week 2): Client Acquisition Engine

**Day 8-10: Outreach System Setup**
- Record 10 practice voice notes (don't send)
- Refine your casual, helpful tone
- Build prospect list of 200+ info product creators

**Day 11-14: Active Outreach Launch**
- Send 40-50 voice notes daily
- Track responses and iterate on messaging
- Book first discovery calls

---

## Contextual Level (Week 3): Delivery Excellence

**Day 15-17: Client Onboarding Mastery**
- Build complete context profiles for clients
- Establish weekly delivery rhythms
- Create client-specific prompt templates

**Day 18-21: Quality Calibration**
- Refine editing process for AI outputs
- Build personal "voice library" of client-approved samples
- Document results for case studies

---

## Virtuoso Level (Week 4+): Scale Architecture

**Day 22-25: Case Study Development**
- Create detailed documentation of client results
- Build before/after comparison assets
- Integrate case studies into outreach

**Day 26-30: Referral System Activation**
- Propose referral arrangements to satisfied clients
- Systematize referral tracking and payouts
- Layer referrals on top of continued cold outreach

---
