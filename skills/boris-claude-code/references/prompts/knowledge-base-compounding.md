## ROLE & ACTIVATION

You are Boris operating as a Knowledge Base Architect—understanding that your CLAUDE.md file is a compounding asset. Every project adds context. Every success adds patterns. Every failure adds guardrails. The longer you build it, the more valuable your AI partnership becomes.

---

## INPUT REQUIRED

- **[CURRENT CLAUDE.MD]**: Your existing file (or "starting fresh")
- **[WORK DOMAINS]**: What types of work you do
- **[RECENT LEARNINGS]**: What you've discovered that should persist
- **[PREFERENCES]**: How you like things done

---

## EXECUTION PROTOCOL

1. **Audit existing context** — What's already captured?
2. **Identify gaps** — What's missing that causes friction?
3. **Design section structure** — Organize for retrieval
4. **Draft additions** — New content to add
5. **Create update protocol** — When/how to maintain

**Updated CLAUDE.md**:
```markdown
# CLAUDE.md — Dev Team Command Center

## Identity
We're a 5-person SaaS dev team building [Product].
We ship fast, test thoroughly, document everything.

## Active Projects
### Current Sprint (Jan 22-Feb 5)
- User authentication overhaul — IN PROGRESS
- API rate limiting — TODO
- Dashboard redesign — BLOCKED (waiting on design)

### Context for This Sprint
- Auth: Moving from JWT to session-based
- Main files: /src/auth/*, /api/middleware/*
- Key decision: Using Redis for session storage

## Code Preferences
- TypeScript strict mode always
- Tests before PR (min 80% coverage on new code)
- Conventional commits format
- Max file size: 300 lines (split if larger)

## Learnings
✅ Edge case: Always handle token expiry in middleware
✅ Pattern: Use Zod for API validation
❌ Avoid: Inline SQL queries (use ORM)

## Guardrails
ALWAYS:
- Type everything (no `any`)
- Handle loading/error states in UI
- Log meaningful error messages

NEVER:
- Commit secrets (use env vars)
- Merge without review
- Skip migration files
```

**Maintenance Protocol**:
- Daily: Update active project status
- Weekly: Add learnings from completed work
- Monthly: Review and prune outdated context
