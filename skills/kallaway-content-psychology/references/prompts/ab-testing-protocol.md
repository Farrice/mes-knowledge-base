## TESTING FUNDAMENTALS

### What You Can Test

| Element | Test Variations | Success Metric |
|---------|-----------------|----------------|
| **Hooks** | Different opening structures | Retention at 0-30 sec |
| **Thumbnails** | Visual concepts, colors, faces | CTR |
| **Titles** | Word choices, formats, lengths | CTR |
| **Video Length** | Short vs. long versions | AVD, completion rate |
| **Posting Time** | Different days/hours | Initial velocity |
| **Content Format** | Talking head vs. B-roll, etc. | Engagement rate |
| **CTA Type** | Different asks, placements | Conversion rate |

### Testing Rules

1. **Test ONE variable at a time** — Otherwise you can't attribute results
2. **Sufficient sample** — Need 3-5 results minimum per variation
3. **Same conditions** — Control for time, audience, etc.
4. **Pre-define success** — Know win criteria before testing
5. **Log everything** — Document results systematically

## TESTING FRAMEWORKS

### Framework 1: The Hook Test

**Setup**:
- Create 5 different hook structures
- Deploy each hook on 3 different videos
- Measure 30-second retention for each

**Hook Types to Test**:
1. Question hook: "Have you ever wondered why..."
2. Bold claim hook: "This is the only method that..."
3. Story hook: "Last week something happened..."
4. Statistic hook: "97% of people fail at..."
5. Direct hook: "Here's exactly how to..."

**Measurement**:
| Hook Type | Video 1 | Video 2 | Video 3 | Average |
|-----------|---------|---------|---------|---------|
| Question | 45% | 52% | 48% | 48.3% |
| Bold claim | 55% | 58% | 52% | 55% |
| ... | ... | ... | ... | ... |

### Framework 2: The Thumbnail Test

**Setup**:
- Create 3 thumbnail concepts for same video
- Use YouTube's or platform's native A/B testing
- Run for minimum 48-72 hours each

**Thumbnail Variables**:
1. Face emotion (shocked vs. happy vs. confident)
2. Text vs. no text
3. Color scheme (bright vs. dark)
4. Composition (close-up vs. wide)

### Framework 3: The Format Test

**Setup**:
- Create same content in 2-3 different formats
- Publish across similar timeframes
- Compare engagement metrics

**Format Options**:
1. Talking head only
2. Talking head + B-roll
3. Screen share + voiceover
4. Text-on-screen (silent watchable)

---

## TESTING CALENDAR

### 4-Week Testing Sprint

**Week 1: Baseline + Hook Test**
- Document current metrics (baseline)
- Begin hook testing (5 variations × 1 video each)

**Week 2: Continue Hook + Start Thumbnail**
- Complete hook testing
- Begin thumbnail testing on winners

**Week 3: Analyze + Format Test**
- Analyze hook/thumbnail results
- Begin format testing

**Week 4: Implement + Iterate**
- Implement winning combinations
- Plan next test cycle

- Platform: YouTube
- Current Metrics: 4% CTR, 45% AVD, 2% subscriber rate

---

### 4-WEEK TESTING PLAN

**Week 1-2: Thumbnail Testing**
- Test 3 thumbnail styles (face, text-heavy, curiosity object)
- 2 videos per style = 6 videos
- Measure CTR after 7 days each

**Week 3: Hook Testing**
- Use winning thumbnail style
- Test 2 hook types on 2 videos each
- Measure 30-second retention

**Week 4: Implementation**
- Combine winning thumbnail + hook
- Measure full metrics
- Document for future content
