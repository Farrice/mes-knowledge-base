## THUMBNAIL PSYCHOLOGY PRINCIPLES

### The 3-Second Rule
Thumbnails are processed in under 3 seconds. Everything must communicate INSTANTLY:
- **Emotion** in 0.5 seconds
- **Topic** in 1 second  
- **Curiosity gap** in 2 seconds

### Visual Hierarchy
1. **Face/emotion** (if present) — Draws eye first
2. **Contrast element** — Creates visual anchor
3. **Text** (if any) — Communicates context
4. **Background** — Sets scene without distracting

### Click Psychology Triggers

| Trigger | Why It Works | Example |
|---------|--------------|---------|
| **Emotion** | Mirror neurons activate | Shock, joy, disgust faces |
| **Incompleteness** | Brain seeks closure | Half-revealed information |
| **Contrast** | Visual pattern interrupt | Bright vs. dark split |
| **Specificity** | Credibility signal | "$47,293" vs. "money" |
| **Before/After** | Transformation curiosity | Split screen |
| **Social Proof** | Trust transfer | Crowd, logos, numbers |

## THUMBNAIL FORMULA TEMPLATES

### Template 1: Reaction Face
```
[EMOTION FACE] + [CONTEXT ELEMENT] + [MINIMAL TEXT]
- Face takes 60% of frame
- Add visual context (what caused reaction)
- Text: 1-3 words maximum
```

### Template 2: Before/After
```
[LEFT: BEFORE STATE] | [RIGHT: AFTER STATE]
- Clear visual contrast
- Arrow or dividing line
- No text needed if contrast is obvious
```

### Template 3: Curiosity Object
```
[UNUSUAL/UNEXPECTED OBJECT] + [QUESTION INDICATOR]
- Object is clearly visible
- Face looking at object
- Creates "what is that?" response
```

### Template 4: Text-Heavy Outcome
```
[BOLD NUMBER/CLAIM] + [SUPPORTING VISUAL]
- Number is largest element
- Visual supports credibility
- Maximum 3 words
```

### Template 5: Story Scene
```
[DRAMATIC MOMENT] + [CONTEXT CLUES]
- Action frozen mid-moment
- Visible stakes/tension
- Makes viewer want to know what happens
```
