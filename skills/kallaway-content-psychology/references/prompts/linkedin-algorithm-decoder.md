## LINKEDIN-SPECIFIC STRATEGIES

### The Algorithm Priorities

1. **Connections First**: Your 1st connections see you first
2. **Dwell Time Matters**: Long reads > quick likes
3. **Comment Begets Comments**: Engagement breeds engagement
4. **Links Are Suppressed**: Put links in comments
5. **Early Engagement Critical**: First hour determines reach

### Text Post Architecture

```
LINE 1: HOOK (Bold statement, question, or pattern interrupt)
[Blank line]
LINE 2-3: CONTEXT (Set up the insight)
[Blank line]
LINE 4-10: BODY (The meat of the insight)
[Blank line]
LINE 11-12: TAKEAWAY (The lesson)
[Blank line]
LINE 13: CTA or QUESTION (Drive comments)
```

**Hook Types That Work on LinkedIn**:
1. **Contrarian**: "Everyone says X. They're wrong."
2. **Personal Failure**: "I [big failure]. Here's what I learned."
3. **Behind the Curtain**: "Most people don't know this about [industry]..."
4. **Specific Number**: "I've done X 500 times. Here's the pattern."
5. **Direct Challenge**: "Stop doing [common practice]."

### Document/Carousel Best Practices

| Element | Optimization |
|---------|--------------|
| **Slides** | 8-12 optimal |
| **First Slide** | Hook that demands swipe |
| **One Idea Per Slide** | Don't overwhelm |
| **Visual Consistency** | Same fonts, colors |
| **Last Slide** | CTA + follow prompt |
