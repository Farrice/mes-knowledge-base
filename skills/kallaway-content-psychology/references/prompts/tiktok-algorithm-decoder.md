## TIKTOK-SPECIFIC STRATEGIES

### The For You Page Algorithm

TikTok serves content in testing pools:
1. **Pool 1**: 200-500 views → If strong metrics → 
2. **Pool 2**: 1,000-5,000 views → If strong → 
3. **Pool 3**: 10,000-50,000 views → If strong → 
4. **Viral Pool**: 100K+ views

**Key Insight**: You have to WIN the small pools before getting promoted.

### Hook Architecture (TikTok-Specific)

| Second | Element | Purpose |
|--------|---------|---------|
| 0.0-0.5 | **STOP** | Pattern interrupt |
| 0.5-1.5 | **HOOK** | Curiosity injection |
| 1.5-3.0 | **SETUP** | Stakes + promise |
| 3.0-end | **DELIVER** | Value + payoff |

**TikTok Hook Types**:
1. **Direct Address**: "If you're [identity], you need to know..."
2. **Controversy Bait**: "I'm about to piss off a lot of [group]..."
3. **Transformation**: Starting with result, then "let me show you how"
4. **Pattern Interrupt**: Unexpected visual/sound start
5. **Trend Participation**: Riding existing momentum

### Optimal Video Specs

| Element | Spec |
|---------|------|
| **Length** | 15-30 seconds optimal for virality |
| **Aspect Ratio** | 9:16 only |
| **Text** | On-screen captions essential |
| **Audio** | Trending sounds = discovery boost |
| **Posting Time** | When YOUR audience is active |
