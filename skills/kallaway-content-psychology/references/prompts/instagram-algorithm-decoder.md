## THE CORE EQUATION

```
Instagram Success = Hook Speed × Completion Rate × Engagement Velocity × Saves/Shares

Reels: First 0.5 seconds determines 80% of performance
Carousels: Swipe-through rate is the key metric
Stories: Completion rate and reply rate
```

## INSTAGRAM-SPECIFIC METRICS

| Metric | What It Means | Target |
|--------|--------------|--------|
| **Reach** | Unique accounts that saw content | Growth indicator |
| **Engagement Rate** | (Likes + Comments + Saves + Shares) / Reach | 3-6% good, 6%+ excellent |
| **Save Rate** | Saves / Reach | 2%+ indicates valuable content |
| **Share Rate** | Shares / Reach | 1%+ indicates viral potential |
| **Watch Time** | % of Reel watched | 80%+ for algorithm favor |
| **Replay Rate** | Times rewatched | Loop potential indicator |

## FORMAT-SPECIFIC STRATEGIES

### REELS (Primary Growth Format)

| Element | Optimization |
|---------|--------------|
| **Length** | 7-15 seconds for reach, 30-60 for depth |
| **First Frame** | Must hook in 0.5 seconds |
| **Loop Design** | End should connect to beginning |
| **Text On Screen** | Essential—many watch muted |
| **Trending Audio** | Use for discovery boost |
| **Vertical Native** | 9:16 ratio only |

**Reels Hook Types**:
1. **Pattern Interrupt**: Unexpected visual start
2. **Question**: "Why do/don't you..."
3. **Controversy**: "Unpopular opinion..."
4. **Transformation**: Before/after reveal
5. **List Tease**: "3 things you need to know about..."

### CAROUSELS (Engagement + Saves Format)

| Element | Optimization |
|---------|--------------|
| **Optimal Length** | 5-10 slides |
| **First Slide** | Hook that demands swipe |
| **Each Slide** | One idea, incomplete loop |
| **Last Slide** | CTA + recap |
| **Consistency** | Same design template |

### STORIES (Relationship Format)

| Element | Optimization |
|---------|--------------|
| **Frequency** | 3-7 per day |
| **Interactive** | Polls, questions, sliders |
| **Behind-Scenes** | Personal content |
| **Links** | Drive to offers/content |

## INSTAGRAM CONTENT ARCHITECTURE

**The 4-2-1 Content Mix** (per week):
- **4 Reels**: Reach and growth

**Content Pillars for Instagram**:
1. **Educational**: Tips, how-tos, frameworks
2. **Inspirational**: Quotes, stories, motivation
3. **Personal**: Behind-scenes, day-in-life
