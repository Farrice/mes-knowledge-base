## Execution

### Step 1: Domain Analysis
Map the current landscape:
- Who are the current KPIs?
- What game are they playing?
- Where are the gaps?
- What's underserved?
- Where's the opportunity?

Provide: **Domain Landscape Analysis**.

### Step 2: Game Definition
If you can't win the current game, create a new one:
- Define a new category
- Name the category
- Establish category criteria
- Position yourself as founder/leader

Provide: **New Game Framework** (or existing game entry strategy).

### Step 3: 5-P Rapid Deployment
Execute all 5 P's simultaneously:
- **Pitch**: Category-defining pitch
- **Publish**: Authority content launch
- **Product**: Quick-win offer
- **Profile**: Credibility sprint
- **Partnership**: Strategic alliances

Provide: **5-P Deployment Plan** with 30-day priorities.

### Step 4: Visibility Blitz
Design 11/90 visibility campaign:
- Content calendar
- Channel strategy
- Collaboration targets
- Amplification tactics

Provide: **Visibility Blitz Plan**.

### Step 5: Demand Engineering
Apply oversubscribed principles in new domain:
- Capacity declaration
- Scarcity architecture
- Waitlist design
- Social proof accumulation

Provide: **New Domain Demand System**.

### Step 6: 90-Day Domination Roadmap
Sequence everything into actionable timeline:
- Week 1-4: Foundation + First visibility
- Week 5-8: Authority building + First clients
- Week 9-12: Acceleration + KPI recognition

Provide: **Complete 90-Day Implementation Calendar**.

## The Fame Game Principle

If you can't win the existing game, change the game:
- "Marketing consultant" → too crowded
- "AI-Powered Revenue Systems Architect for Professional Services" → you just created

The founder of a category is automatically its leader.
