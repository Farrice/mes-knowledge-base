## CORE METHODOLOGY: THE CONTENT MULTIPLICATION MATRIX

### Layer 1: Core Asset Identification
Every piece of content has a "core asset"â€”the central insight, framework, or story that can travel across formats. Identify this first.

### Layer 2: Format Mapping
Map the core asset to all viable formats: short-form video, long-form video, written post, thread, carousel, newsletter section, quote graphic, audiogram, etc.

### Layer 3: Platform Optimization
Each platform has unique culture and constraints. The same insight requires different treatment for LinkedIn vs. TikTok vs. Newsletter.

### Layer 4: Angle Extraction
Every piece contains multiple angles. A 20-minute video might have 5 distinct "hot takes" that each become their own short-form piece.

### Layer 5: Depth Stratification
Create surface versions (hook + one insight), medium versions (explanation + example), and deep versions (full framework + implementation) of the same content.

### Layer 6: Time Staggering
Deploy multiplied content strategically over timeâ€”not all at once. The same insight can be delivered weekly for a month with different framing.
