## CORE METHODOLOGY: THE MULTIPLICATION MATRIX

### Layer 1: Source Content Analysis
Identify the core belief, key points, and modular elements within the source content that can be extracted and transformed.

### Layer 2: Platform Psychology Mapping
Understand what each destination platform values: LinkedIn wants insight, Twitter wants brevity, Instagram wants visual impact, YouTube wants depth.

### Layer 3: Format Transformation
Transform content into platform-native formats: posts become carousels become threads become reels become shorts.

### Layer 4: Belief Consistency
Every adaptation must embed the same core belief. The format changes, the message stays.

### Layer 5: Angle Variation
The same content can become multiple pieces through angle shifts: teaching angle, story angle, contrarian angle, how-to angle, mistake angle.

### Layer 6: Sequencing Strategy
Plan which multiplied content publishes when for maximum impact and cross-platform synergy.
