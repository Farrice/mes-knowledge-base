# Memorable Hooks Forge

Apply Eric Roth's Memorable Residue methodology to engineered hooks and lines. This isn't about "catchy" — it's about creating moments that lodge in memory and become part of how people think and speak.

## The Roth Memorability Framework

Roth's memorable lines work because they combine **specificity** with **universality** — the particular detail that unlocks the general truth. "Life is like a box of chocolates" works because it's childishly simple AND profoundly true. It's fanciful AND real.

### The 5 Residue Mechanisms

| Mechanism | How It Works | Example |
|----------|-------------|---------|
| **Ironic Juxtaposition** | Place the naive next to the tragic, or the mundane next to the epic | "Mama always said, dying was a part of life. I sure wish it wasn't." |
| **Simple Language, Complex Truth** | State something profound in kindergarten vocabulary | "Stupid is as stupid does." |
| **The Reversal** | The second half contradicts the first half's expectation | "I'm not a smart man, but I know what love is." |
| **Sensory Specificity** | One concrete detail that carries an abstract truth | "She tasted like cigarettes." |
| **The Rhythmic Close** | End on a rhythmic beat — iambic or trochaic — that feels like closure | "I may not be a smart man... but I know what love is." (da-DUM da-DUM da-DUM) |

## Execution

### Phase 1: Define the Residue Target

1. **What truth needs to lodge?** State it plainly. (This is the version you'll never use.)
2. **Who needs to remember it?** Different audiences retain different structures.
3. **Where does it live?** Hook (opening), anchor (middle), or close (ending)?
4. **What's its function?** Social sharing, brand recall, emotional crystallization, or worldview shift?

### Phase 2: Generate 10+ Candidates

For each truth, generate at least 10 candidate lines using the 5 Residue Mechanisms:

1. Two Ironic Juxtapositions
2. Two Simple Language/Complex Truth
3. Two Reversals
4. Two with Sensory Specificity
5. Two with Rhythmic Closes

**Rules**:
- No line longer than 15 words (compression creates memorability)
- Each must work standalone — no context needed
- Each must feel like something a human would say, not write
- At least 3 should feel slightly uncomfortable (residue lives at the edge of comfort)

### Phase 3: The Fanciful Real Filter

Test each candidate against Pattern 6:
- Is it emotionally true even if literally impossible?
- Could someone feel this is true even if they can't prove it?
- Does it make you want to nod slightly when you read it?

Cut any line that's clever but not true. Cut any line that's true but not memorable. Keep only lines that are both.

### Phase 4: Contextual Integration

Place the winning lines into their target context:
- **As social hooks**: Does it stop the scroll without clickbait energy?
- **As email subject lines**: Does it create curiosity through truth, not mystery?
- **As landing page headlines**: Does it summarize the offer without sounding like marketing?
- **As taglines**: Does it feel inevitable — like it was always the line?
- **As closing lines**: Does it create the sense of an ending?

### Phase 5: The Tomorrow Test

Final filter — for each candidate, ask:
> If someone read this today, would they remember the exact words tomorrow without looking?

If yes: keep. If they'd remember the *idea* but not the *words*: rework for precision. If neither: cut.

## Output Requirements

**Deliver**:
1. **Truth statement** — the plain version you'll never use
2. **10+ candidates** — labeled by mechanism type
3. **Top 3 finalists** — with Fanciful Real scores and rhythm analysis
4. **The winner** — with contextual placement recommendation
5. **2 alternates** — for different platforms or contexts

**Quality Standard**: The winning line should feel like it was discovered, not written. It should sound like something that was always true but nobody had said yet.
