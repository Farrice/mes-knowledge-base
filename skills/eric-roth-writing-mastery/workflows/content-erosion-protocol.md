# Content Erosion Protocol

Apply Eric Roth's 60-year daily revision discipline to content writing. This is NOT rewriting — it's erosion. Like water shaping stone, each re-entry pass refines the draft without the self-consciousness of "fixing."

## Execution

### Requirements

The user provides **a draft** — blog post, newsletter, article, essay, ghostwritten piece. This workflow deepens it through multiple erosion passes.

### Pass 1: The Full Re-Entry

Read the entire draft from the first word. Don't take notes. Don't plan fixes. Just read. Mark only:
- Where your attention *drifts* (these spots are "falling down")
- Where something *catches* (these spots have latent energy)
- Where the theme feels present vs. absent

**Output**: A diagnostic noting drift points and energy points, mapped against the piece's theme compass.

### Pass 2: Shore Up What's Falling

Return to the first drift point. Don't rewrite — **shore up**. This means:
- Flat sentences → add one sensory detail (visual prose)
- Dead transitions → compress or add white space
- Exposition → displace off-center
- Redundancy → erode (remove the weaker twin)

Move through sequentially. Do not skip ahead.

### Pass 3: Make Tired Things Alive

Now return to the beginning again. This time, focus on:
- **Opening**: Does it invite the reader home? Apply Pattern 12 (Home Invitation). If the opening explains what the piece is about, replace it with a sensory scene.
- **Transitions**: Each section transition should feel like a scene cut, not a logical connector.
- **Closing**: Does it have memorable residue? If not, engineer one line that stays.

### Pass 4: Theme Alignment

Final re-entry from line one. This time, apply the Theme Compass:
- Articulate the piece's theme in one sentence
- For each paragraph, ask: "How does this relate to the theme?"
- Paragraphs with no theme connection: rework or cut
- Paragraphs with strong theme connection: ensure they're not being preachy about it

### Pass 5: The Residue Check

Read the piece one final time and identify:
- [ ] The one line a reader will remember tomorrow
- [ ] The one image that will stay with them
- [ ] The one insight that arrives through the story, not through a thesis statement

If any of these are missing, engineer them using Patterns 9, 7, and 5.

## Output Requirements

**Deliver**:
1. **Erosion diagnostic** — drift/energy map from Pass 1
2. **Annotated revision** — changes made in Passes 2-4 with brief rationale
3. **Final eroded draft** — the polished piece
4. **Residue audit** — the memorable line, image, and insight identified in Pass 5

**Quality Standard**: The opening should be 5x more polished than the closing — evidence that erosion has been at work. The piece should feel like it was written slowly, even if the erosion took 20 minutes.

## Content Type Adaptations

| Content Type | Erosion Focus |
|-------------|--------------|
| **Blog Post** | Heavy visual prose in opening; theme compass across all sections |
| **Newsletter** | Home invitation opening; conversational erosion (tighten voice, not formality) |
| **Long-form Article** | Full 5-pass treatment; most benefit from the process |
| **Ghostwritten Piece** | Erosion must preserve client voice — shore up their patterns, don't introduce yours |
| **LinkedIn Article** | Lighter erosion — 3 passes max; focus on opening and closing residue |
