## How to Deploy a Prompt

1. Navigate to `references/prompts/[prompt-name].md`
2. Copy the complete prompt markdown
3. Provide the required [INPUT] context
4. Execute for comprehensive deliverable

All prompts follow the **MES 3.0 Practitioner Mode Architecture**:
- Role & Activation (expert persona)
- Input Required (what to provide)
- Execution Protocol (step-by-step methodology)
- Output Deliverable (what you'll receive)
- Creative Latitude (when to go beyond the script)
- Enhancement Layer (what makes this special)
- Example Outputs (500+ word demonstrations)
