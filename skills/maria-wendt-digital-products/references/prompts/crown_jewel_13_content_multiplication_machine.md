## THE REPURPOSING HIERARCHY

### EASY Transformations (Do These First)
```
Caption ‚Üí Carousel (Take text, break into slides)
Carousel ‚Üí Story (Post individual slides to stories)
Any Content ‚Üí Repost (Duplicate, wait 60 days, post again)
```

### MEDIUM Transformations
```
Reel Script ‚Üí Carousel (Same content, visual format)
Reel ‚Üí Story Teaser (Clip or reference pointing to reel)
Screenshot from Reel ‚Üí Carousel Slide
```

### HARDER Transformations (More Effort)
```
Carousel ‚Üí Reel (Requires video creation)
Written Content ‚Üí Reel (Requires recording)
```

### THE GOLDEN RULE
Always work DOWN the difficulty ladder, not up. It's easy to turn a reel into a carousel. It's hard to turn a carousel into a reel.
