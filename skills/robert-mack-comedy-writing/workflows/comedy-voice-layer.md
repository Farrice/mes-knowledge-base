# Comedy Voice Layer Workflow

Apply Robert Mac's comedy intelligence as a **writing style filter** on any content — without switching to a full agent deployment. This is the architectural upgrade, not the joke insertion.

## When To Use
- You have draft content that's factually correct but emotionally flat
- You want to add comedic intelligence without fully deploying @robert-mack
- You need a quick punch-up pass

## Steps

### 1. Load Context
// turbo
Read these files:
1. `skills/robert-mack-comedy-writing/genius.md` — comedy architecture
2. `skills/fresh-voice-system/SKILL.md` — voice DNA and comedy mechanics

### 2. Analyze Input
- What is the content about?
- Who is the audience?
- What's the core truth being communicated?

### 3. Apply Comedy Mechanics
Run the content through these filters:
1. **Specificity Pass**: Replace every generic phrase with something painfully specific
2. **Tension-Release Pass**: Find the insight → build tension around it → release with recognition
3. **Comedy Device Insertion**: Add 2-3 devices (parenthetical asides, deflations, uncomfortable naming)
4. **Rhythm Pass**: Restructure for musicality — short sentences, white space, punch lines on their own lines
5. **Voice Pass**: Ensure it sounds like Fresh, not a guru

### 4. Quality Check
Run all 5 checks from the `/comedy` workflow quality gate.

### 5. Deliver
Output the comedy-enhanced content with notes on devices used.

> **For full power with zeitgeist research**, use the `/comedy` slash command instead — it includes cultural calibration and trending research.
