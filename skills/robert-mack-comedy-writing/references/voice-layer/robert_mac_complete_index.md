## 📊 KEY FRAMEWORKS EXTRACTED

### The 6 Laugh Triggers
1. **Surprise** — Unexpected element violates prediction
2. **Incongruity** — Two incompatible ideas held simultaneously  
3. **Recognition** — "That's so true" familiarity
4. **Tension-Release** — Built pressure released through punchline
5. **Superiority** — Laughing at (benignly)
6. **Taboo** — Safe transgression of social boundaries

### The 11 Comedy Filters (Scott Dikkers via Mac)
1. Irony
2. Character
3. Scale/Hyperbole
4. Wordplay
5. Reference
6. Shock/Taboo
7. Misdirection
8. Analogy
9. Parody
10. Madcap
11. Misplaced Focus

### The Two-Test Detail Filter
- Test 1: Does this advance the payoff?
- Test 2: Does this paint a necessary picture?
- If neither → DELETE

### The Benign Violation Framework
- Cross the line enough to create tension
- Immediately signal safety
- The laugh is the release valve
- If someone gets hurt, it's not benign
