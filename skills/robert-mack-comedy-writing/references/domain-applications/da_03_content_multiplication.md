## CORE METHODOLOGIES ENCODED

### The Content Multiplication Formula

**ONE INSIGHT × MULTIPLE FILTERS × MULTIPLE FORMATS = INFINITE CONTENT**

### The 11 Comedy Filters (Applied to Content)

Each filter transforms the same insight into different content:

**1. IRONY FILTER**
- Take the opposite of what the insight seems to say
- "Productivity systems make you less productive" → content about the paradox
- Creates contrarian content that sparks debate

**2. CHARACTER FILTER**
- Voice the insight through different personas
- Expert version / Beginner version / Skeptic version / Enthusiast version
- Same insight, multiple content pieces

**3. SCALE/HYPERBOLE FILTER**
- Exaggerate the insight to extremes
- "This small thing... is actually EVERYTHING"
- Creates dramatic, shareable content

**4. SPECIFICITY FILTER**
- Take the insight and make it hyper-specific
- General insight → specific industry application → specific role → specific moment
- Each specificity level = new content piece

**5. ANALOGY FILTER**
- Connect the insight to unrelated domains
- "Copywriting is like cooking" / "...like dating" / "...like jazz"
- Each analogy = new content with new audience bridge

**6. REVERSAL FILTER**
- Flip the expected order or application
- "Instead of X then Y, what if Y then X?"
- Creates "wait, what?" attention-grabbing content

**7. LIST FILTER**
- Break insight into numbered components
- "3 ways..." / "7 signs..." / "10 reasons..."
- Platform-optimized, highly shareable

**8. STORY FILTER**
- Embed insight in narrative
- Origin story / Failure story / Client story / "This morning..." story
- Each story = new emotional entry point

**9. VERSUS FILTER**
- Position insight against common alternative
- "X vs Y: Why [insight] wins"
- Creates debate and engagement

**10. META FILTER**
- Comment on the insight creation process itself
- "Why I'm sharing this" / "What I almost didn't say"
- Creates authenticity content

**11. PREDICTION FILTER**
- Project insight into future implications
- "In 5 years, everyone will..." / "The brands that survive will..."
- Creates thought leadership content

## DEPLOYMENT PROTOCOL

### Phase 1: Insight Extraction (5 min)

1. **State the Core Insight**
   - One sentence, crystal clear
   - What's the underlying truth?
   - What should someone REMEMBER?

2. **Identify the Tension**
   - What common belief does this challenge?
   - What gap does this reveal?
   - What conflict makes this interesting?

3. **Find the Recognition Hook**
   - What experience does this NAME?
   - What will make audience think "that's so true"?

### Phase 2: Filter Application (15 min)

4. **Apply Each Relevant Filter**:
   - Not all filters work for all insights
   - Select 5-7 that fit naturally
   - For each filter: generate 2-3 angle variations

5. **Generate Headlines/Hooks for Each**:
   - Each filtered version gets its own hook
   - These hooks become content titles/openings

### Phase 3: Format Distribution (10 min)

6. **Map Filters to Platforms**:
   - Which filtered versions work where?
   - Thread-worthy vs. single-post-worthy
   - Visual-ready vs. text-only

7. **Create Content Calendar Slots**:
   - Spread similar content across time
   - Vary formats to avoid repetition feeling
   - Build toward key messages

### Phase 4: Execution (Ongoing)

8. **Produce Content From Matrix**:
   - Each cell = one piece of content
   - Use hooks generated in Phase 2
   - Apply platform-specific optimization
