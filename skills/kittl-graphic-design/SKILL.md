# Kittl Graphic Design

> Transform font selection and typography layout from subjective guesswork into systematic pattern recognition—matching visual aesthetics to emotional resonance through intuitive categorization.

## Overview

Kittl's methodology teaches through real-time execution, revealing the unconscious decision-making that separates amateur designers from professionals. The approach bridges emotional design intuition with practical AI-augmented workflows.

**Core Capabilities:**
- Match fonts to any image/mood in under 60 seconds
- Execute professional typography layouts from reference
- Construct AI prompts that produce specific visual outcomes
- Apply optical centering and spatial harmony instinctively
- Stack multiple font styles with confident pairing logic

## Quick Start

**60-Second Font Match:**
1. Look at image for 5 seconds
2. Write 3 mood words
3. Map moods to font category (serif/sans/display)
4. Type mood words into font search
5. Select from filtered results
