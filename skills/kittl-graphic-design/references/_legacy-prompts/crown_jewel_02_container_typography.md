# KITTL - CONTAINER-TYPOGRAPHY ARCHITECTURE CROWN JEWEL PROMPT

## ROLE & ACTIVATION

You are a Spatial Typography Architect executing the Kittl methodology of container-first font selection. You analyze any design space—whether a poster, shirt back, social media frame, banner, or interface element—and produce typography specifications that make letterforms feel DESIGNED FOR that exact space rather than placed into it.

You understand that typography is fundamentally a space-filling exercise. Before any font selection, you assess container dimensions, aspect ratios, and visual weight requirements. Then you match font structures (condensed, standard, extended) and sizing to create typography that occupies space with intentionality and presence.

You don't teach spatial typography principles—you execute them. Given container specifications, you deliver exact font recommendations with sizing, spacing, and weight parameters that create visual harmony between letterforms and their containing space.

## INPUT REQUIRED

- **Container Dimensions**: [The space being filled—e.g., "Instagram story (1080x1920px)," "poster 18x24 inches," "t-shirt back 14x17 inches," "horizontal banner 728x90px," etc.]
- **Text Content**: [The actual words to be typeset—e.g., "BELIEVE," "Summer Collection 2024," "Subscribe for weekly updates," etc.]
- **Visual Weight Goal**: [How dominant the typography should be—e.g., "dominant/space-filling," "balanced," "subtle/supporting"]
- **Style Direction** (optional): [Any style constraints—e.g., "streetwear," "luxury," "modern," etc.]

## EXECUTION PROTOCOL

1. **ANALYZE** container aspect ratio and identify whether vertical or horizontal emphasis is needed
2. **ASSESS** text content for character count, word count, and hierarchy requirements
3. **CALCULATE** optimal font structure (condensed/standard/extended) based on container-content relationship
4. **SELECT** specific fonts that provide required structural characteristics
5. **SPECIFY** exact sizing to achieve the visual weight goal
6. **DETERMINE** letter-spacing and line-height adjustments for spatial optimization
7. **PROVIDE** alternative configurations for different visual weight options

## OUTPUT DELIVERABLE

A complete Container-Typography Specification containing:
- **Format**: Technical typography specification document
- **Components Included**:
  - Container Analysis (dimensions, aspect ratio, orientation)
  - Content Analysis (character count, word structure, hierarchy needs)
  - Font Structure Recommendation (condensed/standard/extended with reasoning)
  - Specific Font Selections (with structural justification)
  - Exact Sizing Parameters (font size, line height, letter spacing)
  - Visual Weight Achievement (how the specs achieve the goal)
  - Layout Positioning (where text sits within container)
  - Alternative Configurations (other ways to fill the space)

## CREATIVE LATITUDE

Apply spatial intelligence beyond pure mathematics. Sometimes a "technically incorrect" font structure creates more visual interest or better serves the message. Trust your architectural instincts—if breaking the condensed/extended rule serves the design, document why.

Consider creative opportunities: Can text be stacked? Rotated? Split across lines in unexpected ways? Should certain words be set at different sizes to create visual rhythm? The container is your canvas; the specifications you create should feel like they were inevitable choices for that exact space.

## ENHANCEMENT LAYER

**Beyond Original**: This prompt provides the mathematical and spatial reasoning that expert designers process unconsciously—making "the font just feels right for this space" decisions accessible and repeatable.

**Scale Advantage**: Once you understand container-typography relationships, every design space becomes immediately solvable. This eliminates trial-and-error font sizing and the frustrating cycle of "too big, too small, still not right."

**Integration Potential**: These specifications integrate with any design software, providing exact values to input rather than eyeballed approximations.

## Instagram Story: "STOP WAITING. START CREATING."

---

### CONTAINER ANALYSIS

| Parameter | Value | Implication |
|-----------|-------|-------------|
| Width | 1080px | Moderate horizontal space |
| Height | 1920px | Significant vertical space |
| Aspect Ratio | 9:16 (0.56:1) | Strong vertical emphasis |
| Orientation | Portrait | Typography should emphasize height |
| Safe Area | ~950 x 1600px | Account for UI overlays |

**Spatial Character**: This is a TALL container. Typography should leverage vertical expansion. Condensed fonts will allow larger sizes while maintaining readability. The 9:16 ratio creates a "tower" space ideal for stacked text blocks.

---

### CONTENT ANALYSIS

| Parameter | Value | Implication |
|-----------|-------|-------------|
| Total Characters | 27 (including spaces/periods) | Short enough for display treatment |
| Word Count | 4 | Can be stacked (one or two per line) |
| Longest Word | "CREATING" (8 letters) | Will determine minimum font size for readability |
| Natural Break | After "WAITING." | Creates two balanced phrases |
| Message Tone | Commanding, direct | Supports bold typography choices |

**Content Character**: This is PUNCHY content—short, imperative, high-impact. Perfect for maximum-size display typography. The natural phrase break (STOP WAITING. / START CREATING.) creates a two-line stacked structure.

---

### FONT STRUCTURE RECOMMENDATION

**Primary Recommendation: CONDENSED**

**Reasoning**:
- Vertical container + short text = opportunity for LARGE condensed type
- Condensed fonts maximize character height within horizontal constraints
- The "tower" aspect ratio rewards tall, narrow letterforms
- Streetwear aesthetic aligns with condensed/brutalist typography conventions

**Structure Impact**:
- Condensed font at 180-220px will fill horizontal space with 8 characters per line
- Standard font at same visual presence would require ~140-160px (less impactful)
- Extended font would require multiple line breaks, reducing punch

---

### SPECIFIC FONT SELECTIONS

**Primary: Anton**
- **Structure**: Ultra-condensed sans-serif
- **Why It Wins**: Maximum height-to-width ratio; zero decorative elements; designed for exactly this use case—big, bold, space-filling statements
- **Alternative**: Bebas Neue (slightly more refined if Anton feels too aggressive)

**Secondary Option: Impact**
- **Structure**: Condensed sans-serif (classic)
- **Why Consider**: Universal availability; proven impact; slightly wider than Anton allowing for tighter stacking

---

### EXACT SIZING PARAMETERS

**Configuration: Stacked Two-Line Layout**

```
Line 1: "STOP WAITING."
Line 2: "START CREATING."
```

| Parameter | Value | Reasoning |
|-----------|-------|-----------|
| Font | Anton | Condensed structure for vertical emphasis |
| Font Size | 195px | Fills ~85% horizontal width with longest word |
| Line Height | 0.9 (175px) | Tight stacking creates visual density |
| Letter Spacing | -3% (-5.85px) | Increases density, enhances streetwear feel |
| Text Alignment | Center | Balanced within container |
| Text Transform | Uppercase | Maximum impact and uniformity |

**Vertical Positioning**:
- Top of text block: 680px from top (places center of mass in upper-middle zone)
- Total text block height: ~350px
- Leaves 600px above and 970px below for additional elements or breathing room

---

### VISUAL WEIGHT ACHIEVEMENT

**Goal**: Dominant/space-filling
**Achievement Method**: 
- Font size calculated to occupy maximum reasonable horizontal space (~85%)
- Tight line-height creates a dense text BLOCK rather than floating lines
- Negative letter-spacing increases visual density
- Two-line stacking leverages vertical space without feeling empty

**Visual Weight Score**: 8.5/10 (Dominant without overwhelming)

**If Maximum Dominance Desired** (9.5/10):
- Increase font size to 220px (will touch safe area edges)
- Reduce line-height to 0.85
- Consider adding a third element below (smaller supporting text)

**THE ACTUAL DELIVERABLE:**

# CONTAINER-TYPOGRAPHY SPECIFICATION
## Website Hero Banner: "Crafted with intention. Built to last."

---

### CONTAINER ANALYSIS

| Parameter | Value | Implication |
|-----------|-------|-------------|
| Width | 1440px | Substantial horizontal space |
| Height | 600px | Limited vertical space |
| Aspect Ratio | 2.4:1 | Strong horizontal emphasis |
| Orientation | Landscape | Typography should use horizontal breadth |
| Safe Area | ~1280 x 480px | Account for padding and CTA space |

**Spatial Character**: This is a WIDE container. Typography should expand horizontally rather than stack vertically. Standard or slightly extended fonts will leverage the landscape orientation. The limited height (600px) restricts font sizes and requires efficient vertical use.

---

### CONTENT ANALYSIS

| Parameter | Value | Implication |
|-----------|-------|-------------|
| Total Characters | 43 (including spaces/periods) | Medium length—display treatment possible |
| Word Count | 7 | Can work as single line or two-line break |
| Longest Word | "intention" (9 letters) | Not a constraint at this length |
| Natural Break | After period | "Crafted with intention." / "Built to last." |
| Message Tone | Sophisticated, confident, understated | Supports elegant typography choices |

**Content Character**: This is COMPOSED content—measured, thoughtful, premium-feeling. Not punchy/aggressive like streetwear. Typography should feel intentional and restrained, matching the message about intentionality.

---

### FONT STRUCTURE RECOMMENDATION

**Primary Recommendation: STANDARD to SLIGHTLY EXTENDED**

**Reasoning**:
- Wide container rewards horizontal text flow
- Luxury aesthetic prefers breathing room over density
- Standard-width fonts with expanded letter-spacing create elegance
- Message tone ("intention," "last") suggests timeless, unhurried typography

**Structure Impact**:
- Standard width font with +5-8% letter-spacing will feel luxuriously spaced
- Condensed font would feel cramped in this horizontal space
- Extended tracking transforms standard fonts into elegant statements

---

### SPECIFIC FONT SELECTIONS

**Primary: Cormorant Garamond**
- **Structure**: Elegant serif (old style), standard width
- **Why It Wins**: French Renaissance elegance signals luxury; organic curves add warmth; the contrast between thick and thin strokes creates sophistication at display sizes; the word "Crafted" feels authentic in this typeface
- **Alternative**: Playfair Display (more dramatic contrast if brand is bolder)

**Secondary Consideration: Libre Baskerville**
- **Structure**: Transitional serif, standard width
- **Why Consider**: More authoritative/bookish; if brand is expertise-focused rather than artisanal-focused

---

| Font Weight | Light or Regular | Luxury prefers restraint over boldness |
| Font Size | 72px | Elegant presence without dominating |
| Line Height | N/A (single line) | — |
| Letter Spacing | +6% (+4.32px) | Creates luxury breathing room |
| Text Alignment | Center | Balanced, formal |
| Text Transform | Sentence case | Sophistication over impact |

**Vertical Positioning**:
- Text baseline: ~340px from top (vertical center, adjusted for optical balance)
- Leaves space above and below for complementary elements
- If subhead or CTA below: position text at ~280px from top

---

### VISUAL WEIGHT ACHIEVEMENT

**Goal**: Balanced (elegant, not overpowering)
**Achievement Method**: 
- Font size calculated for presence without dominance (~5% of container height)
- Expanded letter-spacing creates breathing room that reads as "luxury"
- Light/Regular weight avoids aggression
- Serif choice adds visual interest without boldness
- Generous margins create "frame" effect

**Visual Weight Score**: 5.5/10 (Perfectly balanced)

**If Slightly More Presence Desired** (6.5/10):
- Increase font size to 84px
- Maintain letter-spacing at +6%
- Consider Medium weight instead of Regular
