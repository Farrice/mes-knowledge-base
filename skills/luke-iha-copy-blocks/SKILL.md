# Luke Iha: Copy Blocks & CASH Method

## What This Skill Does

This skill operationalizes the ATOMIC LAYER of Luke Iha's copywriting system — the building blocks from which every ad, VSL, lead, email, and landing page is constructed. While other Luke Iha skills handle strategy (creative-strategy), opening architecture (vsl-leads), and mechanism engineering (million-dollar-mechanisms), THIS skill handles the actual WRITING.

Two core systems:
1. **Copy Blocks** — The 6 elements every persuasive piece uses: Pain, Promise, Proof, Constraints, Curiosity, Conditions. The secret isn't using all 6 — it's how FAST you cycle through them (copy velocity).
2. **CASH Method** — A systematic ideation machine: Concept × Angle × Style × Hook. Generates 100+ ad variations from a single product without creative burnout.

## Available Workflows

| # | Workflow | Produces |
|---|----------|----------|
| 1 | **Copy Block Audit** | Diagnostic of any copy: block map, velocity chart, missing blocks, rewrite recommendations |
| 2 | **CASH Method Generator** | 20+ ad concepts from one product using Concept × Angle × Style × Hook matrix |
| 3 | **Ad Script Writer** | Complete ad script with hook variants, body copy, and CTA — built from copy blocks |
| 4 | **Hook Forge** | 30+ hooks using positioning types, psychological triggers, and proof-of-work patterns |

## When to Use

- Before writing ANY ad, VSL, email, or landing page (run Copy Block Audit on drafts)
- When you need endless ad variations (CASH Method replaces "I'm out of ideas")
- When hooks are underperforming (Hook Forge generates testable alternatives)
- When copy "feels flat" (usually a velocity problem — blocks aren't cycling fast enough)
- When training AI to write copy (Copy Blocks give it structural scaffolding)
