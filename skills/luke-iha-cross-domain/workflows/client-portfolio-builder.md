# Client Portfolio Builder

Construct a comprehensive freelance portfolio that demonstrates mastery across all 6 Luke Iha domains. Uses spec work (rebuilding real brands' assets) to create portfolio pieces that function as Nuclear VSLs — the work itself IS the pitch.

---

## Skills Used

| Skill | Portfolio Piece It Produces |
|-------|---------------------------|
| `creative-strategy` | Creative strategy brief + positioning statement |
| `copy-blocks` | Annotated ad scripts showing copy block mastery |
| `vsl-leads` | Complete VSL lead with structural annotations |
| `million-dollar-mechanisms` | Mechanism discovery brief with validation |
| `unaware-ads` | Unaware ad campaign with awareness-level mapping |
| `proof-mechanisms` | Proof architecture document |

---

## Inputs Required

1. **Target Client Niche** — Who do you want to work for? (e.g., supplement brands, SaaS, coaches)
2. **1-3 Dream Brands** — Specific brands you'd love to work with
3. **Your Agency Ladder Rung** — Where are you now? Where do you want to be?
4. **Category of One Statement** — Your positioning (or we'll build it)

---

## Phase 1: Portfolio Architecture

### 1.1 Piece Selection (6 pieces minimum)

| # | Portfolio Piece | Demonstrates | Format |
|---|----------------|-------------|--------|
| 1 | **Mechanism Brief** | Strategic thinking, research depth | PDF/doc |
| 2 | **VSL Lead** | Long-form copy mastery | Written copy + annotated |
| 3 | **Ad Campaign Suite** | Volume, variety, systematic ideation | 5+ ad scripts + CASH matrix |
| 4 | **Unaware Ad** | Audience psychology, awareness targeting | Script + strategic rationale |
| 5 | **Creative Strategy Brief** | Meta-skill, upstream thinking | Strategic document |
| 6 | **Copy Block Audit** | Analytical skill, diagnostic ability | Audit report of real brand's copy |

### 1.2 Brand Selection

Pick a real brand in your target niche. You'll rebuild their worst asset:

- **Brand**: ___
- **Their weakest asset (ad, landing page, email)**: ___
- **Why it's weak (quick diagnosis)**: ___

---

## Phase 2: Piece Creation

### Piece 1: Mechanism Brief
Run **Mechanism Discovery Engine** for the chosen brand's product.
Annotate with your strategic thinking — *why* you chose UMP/UMS, *why* this SIN score.

### Piece 2: VSL Lead
Run **VSL Lead Writer** using the mechanism from Piece 1.
Include structural annotations showing the 15-element checklist.

### Piece 3: Ad Campaign Suite
Run **CASH Method Generator** → generate 20+ concepts.
Run **Ad Script Writer** → write top 5 as complete scripts.
Include the CASH matrix showing your systematic process.

### Piece 4: Unaware Ad
Run the unaware ads workflow for the brand.
Show the awareness-level reasoning.

### Piece 5: Creative Strategy Brief
Run **Creative Strategy Brief Generator** for the brand.
This demonstrates you think BEFORE you write.

### Piece 6: Copy Block Audit
Take the brand's EXISTING worst ad/page.
Run **Copy Block Audit** → diagnose what's wrong.
Add your recommended rewrite.

---

## Phase 3: Presentation Layer

### For Each Piece:
1. **Before** (the brand's original, if applicable)
2. **After** (your version)
3. **Strategic Rationale** (Loom walkthrough or written annotation explaining *why*)

### Portfolio Landing Page Structure:
```
Hero: "I [positioning statement]"
Section 1: Featured Case Study (best piece, full walkthrough)
Section 2: Portfolio Grid (thumbnails of all 6 pieces)
Section 3: Process Overview (how you work — visual flow)
Section 4: Results/Reviews (if available)
Section 5: CTA ("Let me audit your [asset]")
```

---

## Phase 4: Nuclear VSL Integration

Turn the portfolio itself into a Nuclear VSL:
- The portfolio page IS the sales page
- Each piece demonstrates a different aspect of competence
- The Loom walkthrough functions as a VSL — you're explaining strategy while showcasing work
- CTA offers a free audit (entry offer) that leads to paid engagement
