# VSL-to-Ad Atomizer

Take a single VSL lead and decompose it into 10+ standalone ads. Every section, fascination, micro lead, and proof element in a VSL lead is raw material for independent ads. This workflow systematically extracts and reformats.

---

## Skills Used

| Skill | How It's Used |
|-------|--------------|
| `vsl-leads` | Lead architecture understanding + micro lead types |
| `copy-blocks` | Copy block extraction + ad reformatting |
| `unaware-ads` | Awareness-level targeting for each extracted ad |

---

## Inputs Required

1. **Existing VSL Lead** — Complete lead copy (or lead you've written with VSL Lead Writer)
2. **Product/Offer** — What the lead is selling
3. **Mechanism** — If present, the mechanism name and type

---

## Phase 1: Lead Decomposition

Break the lead into its structural components:

### 1.1 Extract the Micro Lead
- Isolate the first 50-200 words
- Identify which of the 7 types it is
- **Ad potential**: This is already a standalone hook + setup → reformat as a short ad

### 1.2 Extract Fascination Bullets
- Pull every "you'll discover..." or curiosity bullet
- Each fascination = one potential curiosity-driven ad
- **Ad potential**: Each fascination becomes a hook: "[Fascination] — link in bio"

### 1.3 Extract Proof Elements
- Testimonials, data points, case studies, expert citations
- **Ad potential**: Each proof element becomes a proof-of-work or social proof ad

### 1.4 Extract Contrarian Claims
- Statements that challenge conventional wisdom
- **Ad potential**: Each contrarian claim becomes a hook for a problem-aware audience

### 1.5 Extract Future Pacing
- Sections that paint the "after" picture
- **Ad potential**: Aspirational ads targeting solution-aware audiences

### 1.6 Extract Mechanism Reveals
- The point(s) where the mechanism is introduced or explained
- **Ad potential**: Educational/discovery ads targeting unaware audiences

## Phase 2: Atomization Grid

Map each extracted element to an ad format:

| # | Source Element | Type | Awareness Level | Ad Format | Length |
|---|---------------|------|-----------------|-----------|--------|
| 1 | Micro lead | Hook + setup | Problem-aware | Short video script | 30s |
| 2 | Fascination 1 | Curiosity | Solution-aware | Static/carousel | 1 image |
| 3 | Fascination 2 | Curiosity | Solution-aware | Static/carousel | 1 image |
| 4 | Testimonial 1 | Proof | Product-aware | UGC testimonial | 15s |
| 5 | Contrarian claim | Challenge | Unaware | Talking head | 30s |
| 6 | Future pacing | Aspiration | Solution-aware | Voiceover + B-roll | 30s |
| 7 | Mechanism reveal | Discovery | Unaware | Expert explainer | 60s |
| ... | | | | | |

**Target**: 10+ ads from one VSL lead.

---

## Phase 3: Ad Script Writing

For each atomized element, write a complete ad:

### Per-Ad Template:
```
HOOK: [Extracted/adapted from lead element]
BODY: [Reformatted with copy blocks — compress lead section into 30-60 sec]
CTA: [Match to awareness level]
```

### Copy Block Mapping:
For each ad, identify which copy blocks it uses:
- ☐ Pain  ☐ Promise  ☐ Proof  ☐ Constraints  ☐ Curiosity  ☐ Conditions
