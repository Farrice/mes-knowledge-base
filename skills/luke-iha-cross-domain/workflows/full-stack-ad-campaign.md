# Full-Stack Ad Campaign

Build a complete advertising campaign from scratch using the full Luke Iha skill suite. Starts with mechanism engineering, flows through lead architecture, and produces multiple ad variations via the CASH method.

---

## Pipeline

```
Mechanism Discovery → Validation → Little Big Idea
        ↓
Copy Block Architecture → CASH Method
        ↓
VSL Lead → Micro Leads (7 types)
        ↓
Ad Scripts (5+ variations) → Hook Bank
        ↓
Creative Strategy Brief (targeting + positioning)
```

## Skills Used

| Phase | Skill | Workflow |
|-------|-------|---------|
| 1 | `million-dollar-mechanisms` | Mechanism Discovery Engine |
| 2 | `million-dollar-mechanisms` | Mechanism Validator + Little Big Idea Finder |
| 3 | `million-dollar-mechanisms` | Mechanism-to-Copy Bridge |
| 4 | `copy-blocks` | CASH Method Generator |
| 5 | `vsl-leads` | VSL Lead Writer + Micro Lead Generator |
| 6 | `copy-blocks` | Ad Script Writer + Hook Forge |
| 7 | `creative-strategy` | Creative Strategy Brief Generator |

---

## Phase 1: Mechanism Foundation (20 min)

### Inputs:
- Product/offer description
- Target audience
- Competitive landscape

### Execute:
1. Run **Mechanism Discovery Engine** → 10+ candidates
2. Run **Mechanism Validator** on top 2-3 → select winner
3. Run **Little Big Idea Finder** → strategic insight

### Checkpoint:
- ☐ Validated mechanism with SIN ≥ 21
- ☐ Characterization (name) passes dinner table test
- ☐ Little Big Idea expressible without product name

---

## Phase 2: Copy Generation (30 min)

### Execute:
4. Run **Mechanism-to-Copy Bridge** → hooks, lead angles, ad concepts, VSL outline
5. Run **CASH Method Generator** → 20+ ad concepts from mechanism
6. Run **Hook Forge** → 30+ hooks from mechanism insights

### Checkpoint:
- ☐ 17+ mechanism-driven hooks
- ☐ 20+ CASH combinations
- ☐ 5 distinct lead angles

---

## Phase 3: Long-Form Asset (20 min)

### Execute:
7. Run **VSL Lead Writer** → complete lead using strongest micro lead
8. Run **Micro Lead Generator** → 7 testable variants
9. Run **Fascination Bullet Factory** → 30+ fascinations

### Checkpoint:
- ☐ Complete VSL lead with all 15 elements
- ☐ 7 micro lead variants for testing
- ☐ Fascinations create genuine curiosity

---

## Phase 4: Ad Production (20 min)

### Execute:
10. Run **Ad Script Writer** → 5 complete ad scripts using top CASH combinations
11. Select top hooks from Hook Forge for each script
12. Run **Copy Block Audit** on each script → verify block coverage

### Checkpoint:
- ☐ 5 complete ad scripts with block annotations
- ☐ Each script uses at least 4/6 copy blocks
- ☐ Hooks are mechanism-driven (not generic)

---

## Phase 5: Strategic Wrapper (10 min)

### Execute:
13. Run **Creative Strategy Brief** → targeting, positioning, channel recommendations
14. Map ads to awareness levels
15. Define testing sequence

### Final Deliverable:
```
Full-Stack Ad Campaign Package:
├── Mechanism Brief (name, type, SIN score, Little Big Idea)
├── VSL Lead (complete + 7 micro lead variants)
├── Ad Scripts × 5 (with block annotations)
├── Hook Bank (30+ organized by type)
├── Fascination Bullets (30+)
├── CASH Combination Matrix
└── Creative Strategy Brief (targeting + testing plan)
```
