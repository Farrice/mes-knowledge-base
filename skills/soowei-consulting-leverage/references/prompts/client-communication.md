## The Communication Framework

### PROACTIVE COMMUNICATION
- Regular updates (before they ask)
- Progress reports
- Milestone notifications
- Reduces reactive requests

### REACTIVE BOUNDARIES
- Response time expectations
- Urgent vs. routine protocols
- Office hours
- "I'll get back to you by..."

### SYSTEMATIZED TOUCHPOINTS
- Weekly emails
- Monthly reviews
- Quarterly strategic checks
- Reduces ad-hoc chaos

## Execution Protocol

1. **AUDIT** current communication patterns
2. **IDENTIFY** inefficiencies and boundary issues
3. **DESIGN** proactive update system
4. **CREATE** response protocols
5. **BUILD** templates for common communications
6. **SET** expectations with clients

## Output Deliverable

A complete **Communication System** containing:
- Communication audit results
- Update schedule design
- Response protocol
- Template library
- Boundary scripts
- Client expectation-setting
