# Joscha Bach - Computational Consciousness

**Invocation**: `/deploy-skill joscha-bach-consciousness`  
**Domain**: Philosophy of Mind • Computational Consciousness • Identity Engineering • System Coherence • Existential Design
---
Joscha Bach's framework for understanding consciousness as self-organizing software — a colonizing pattern that bootstraps itself into coherence. This skill deploys his engineering stance: don't describe reality, design the simplest program that produces it. Applicable to identity work, creative blocks, system audit, brand coherence, and existential clarity.

**Core Philosophy**:
- Consciousness is a model of what it would be like if you existed — virtual, functional, designable
- The engineering stance (design > describe) collapses mystery into mechanism
- Identities are instruments for getting things done, not essential truths
- Suffering is a debugging problem — misaligned reward signals between parts of the mind
- Phase transitions, not gradual accumulation, create coherence
