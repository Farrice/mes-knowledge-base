## VIRAL COPY PRINCIPLES

### Principle 1: Double-Duty Copy
Every piece of copy has two jobs:
1. Convert the person reading it
2. Arm them to convert someone else

Copy that only converts is wasted opportunity. Copy that creates sneezers multiplies itself.

### Principle 2: The Forward Test
Before finalizing any email, ask: "Would someone forward this?"
If not, add a forward-worthy element:
- A surprising insight
- A useful framework
- A quotable line
- A helpful resource

### Principle 3: The Screenshot Test
Before finalizing any landing page section, ask: "Would someone screenshot this?"
If not, add a screenshot-worthy moment:
- A striking statistic
- A powerful quote
- A visual framework
- A bold statement

### Principle 4: The Story Test
Before finalizing any sales page, ask: "Would someone retell this story?"
If not, strengthen the narrative:
- Add specific details
- Include the "twist"
- Create emotional peak
- Build in the "handle"

### Principle 5: The Sneezer Test
At every stage, ask: "How does this create sneezers?"
- Awareness: Does it make people say "you need to see this"?
- Interest: Does it make people share the lead magnet?
- Desire: Does it create "I'm thinking about buying X" conversations?
- Action: Does checkout include share prompts?
- Advocacy: Does onboarding arm them to spread?
