# Sean Mabry Voice Mastery

> Master voice writing through prediction, not imitation. Build a mental model of how a thought leader thinks, then deploy it across long-form content where every sentence must pass the "sounds like me" test.

## Deployable Capabilities (12 Prompts)

### Phase 1 — Capture & Calibrate
| # | Capability | Use When |
|---|-----------|----------|
| 1 | **Voice Calibration Sprint** | Starting a new engagement — collapse the 6-month calibration period into weeks |
| 2 | **Voice Document Builder** | Building the foundational voice reference that any team member can use |
| 3 | **Hidden Gems Story Miner** | Mining podcast appearances for powerful stories the client doesn't realize they have |
| 4 | **Client Interview Engine** | Conducting structured interviews that extract what the client doesn't know they know |

### Phase 2 — Write & Audit
| # | Capability | Use When |
|---|-----------|----------|
| 5 | **Voice Accuracy Audit** | Scoring any draft against voice fidelity markers — find and fix every voice break |
| 6 | **Story Remix Engine** | Deploying the same story across multiple formats without repetition fatigue |
| 7 | **Memoir Architect** | Planning a thought leader's first book — Authorship Pyramid + hero's journey structure |

### Phase 3 — Scale & Monetize
| # | Capability | Use When |
|---|-----------|----------|
| 8 | **Book-to-Content Atomizer** | Breaking a book into 50+ content pieces across all platforms |
| 9 | **Voice Handoff Kit** | Packaging your voice mastery so a team (or AI) can write at 80%+ fidelity |
| 10 | **Trust Ladder Navigator** | Managing the client relationship from trial through autonomous publishing |

### Phase 4 — Position & Grow
| # | Capability | Use When |
|---|-----------|----------|
| 11 | **Voice Niche Positioner** | Choosing high-sensitivity niches where voice commands premium fees |

## Quick Start Pipelines

**New ghostwriting client (full onboarding):**
1. Load `genius.md` for Mabry's core patterns
2. Deploy Voice Calibration Sprint → capture voice fingerprint
3. Deploy Hidden Gems Story Miner → build story bank
4. Deploy Voice Document Builder → create the single source of truth
5. Deploy Client Interview Engine → extract deeper material
6. Begin prediction practice within Week 2

**Planning a book for a personal brand:**
1. Load `genius.md` + `references/implementation.md`
2. Deploy Memoir Architect → positioning + outline
3. Deploy Client Interview Engine → multi-round story extraction
4. Deploy Book-to-Content Atomizer → plan content flywheel from book material

**Auditing and improving existing voice work:**
1. Deploy Voice Accuracy Audit → score current drafts
2. Review edit patterns → identify recurring voice breaks
3. Deploy Story Remix Engine → get more mileage from existing stories
4. Deploy Voice Handoff Kit → enable team scaling
