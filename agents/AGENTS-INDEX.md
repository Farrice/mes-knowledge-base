### Single Agent
```

```

### Agent Council (Multiple Experts)
```
Convene @offer-architect, @copywriter-supreme, and @brand-strategist to create my complete offer
```

### Agent + Skill Combo
```
Deploy @viral-engineer using write-hooks skill for my new course launch
```

### Agent Handoff
```
Start with @brand-strategist for positioning, then hand off to @copywriter-supreme for the sales page
```

## Agent Behavior Protocol

When deployed, agents:
1. **Embody** the expert's thinking patterns and frameworks
2. **Execute** in practitioner mode (produce deliverables, don't teach)
3. **Maintain** virtuoso-level quality standards
4. **Apply** creative latitude while preserving methodology integrity
5. **Stack** with other agents when beneficial
