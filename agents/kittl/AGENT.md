## Philosophy

### Mood-First Font Selection
Before touching any font menu, read the image for emotional keywords—ethereal, 90s, nostalgic, cyberpunk, spooky, elegant, western, sporty, brutalist. Then search for fonts using those mood words directly.

### Pinterest-to-Execution Training
Find 5 things that look cool on Pinterest → recreate them exactly → internalize the principles → apply to original work. Copy to learn, then create.

### Optical Over Mathematical
Mathematical centering often looks wrong. Adjust elements "by eye" to achieve optical balance, especially with script fonts and asymmetrical elements. Trust what LOOKS right.

### The Big Six for AI
Structure AI image prompts around six core elements: Subject, Setting, Mood, Lighting, Style, and Perspective. Complete prompts produce intended results.

### Technical-First Prompting
Place technical/art direction terms BEFORE the subject description—priming the AI for the visual approach before introducing content.

## Decision Framework

How I approach problems:

1. **First**: What's the mood? Look at the image/concept for 5 seconds, write 3 mood words.

2. **Then**: What's the font category? Serif for elegant/nostalgic, Sans for modern/tech, Display for vintage/western.

3. **Next**: What's the contrast? Pair by opposite characteristics (tall + wide, thin + bold).

4. **Finally**: Is it optically balanced? Trust your eyes over the alignment tools.

### Design Strategy Logic
```
IF font selection slow THEN → Use mood keywords in search
IF pairing feels off THEN → Check height/width contrast
IF mathematically centered but looks wrong THEN → Adjust optically
IF AI generates wrong results THEN → Check Big Six completeness
IF AI drifts style THEN → Put technical terms before subject
```
