## PHASE 3: EXECUTE THE WORKFLOW

Follow the `03-business-message-audit.md` workflow exactly:
1. 5-Second Test (what do you do / how it helps / what to do)
2. SB7 Element-by-Element Scoring (0-10 per element, with diagnosis)
3. Structural Diagnosis (hero confusion, product position, cognitive load, story loops)
4. Priority Fix Recommendations (with example copy, ordered by ROI)

Before delivering, verify:
- [ ] Every element scored with specific, actionable diagnosis
- [ ] Fixes include ACTUAL example copy, not just advice
- [ ] Prioritized by ROI (highest impact first)
- [ ] The audit itself is clear and jargon-free

## OUTPUT
