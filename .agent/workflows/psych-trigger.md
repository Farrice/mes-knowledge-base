# /psych-trigger

Deploy Jeremy Miner's psychological trigger architecture for any persuasive context.

## Usage

```
/psych-trigger
```

## What It Does

Takes a belief you need to shift and builds a complete frame/deframe/reframe sequence with tonal annotations, self-persuasion checkpoints, and deployment context.

## Execution

1. **Read skill files:**
   - `skills/jeremy-miner-identity-persuasion/SKILL.md`
   - `skills/jeremy-miner-identity-persuasion/genius.md` — Focus on Patterns 15, 17, 19, 23, 24, 27, 28
   - `skills/jeremy-miner-identity-persuasion/workflows/psychological-trigger-architect.md`

2. **Gather inputs:**
   - What belief do you need to shift?
   - Who is the audience?
   - What context? (sales call, copy, content, presentation)
   - What's the desired new belief?

3. **Execute the Psychological Trigger Architect workflow** end-to-end

4. **Deliver:**
   - 3-5 complete trigger sequences (frame → deframe → reframe)
   - Tonal annotations on every line
   - Self-persuasion checkpoints
   - Deployment context for each trigger

## Stacks With

- `/build-gap` — build the gap first, then deploy triggers to widen it
- `/prevent-objection` — use triggers as prevention architecture
- `/tone-calibrate` — annotate trigger scripts with full tonal direction
- `/frame-engineer` — dedicated frame engineering (more granular version)
