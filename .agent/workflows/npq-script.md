# /npq-script

Build an end-to-end NEPQ (Neuro-Emotional Persuasion Questions) sales conversation script for any industry.

## Usage

```
/npq-script
```

## What It Does

Generates a complete conversation flow with 8 stages — from nervous system calming opening through self-persuasion close — fully annotated with tonal direction and pacing markers.

## Execution

1. **Read skill files:**
   - `skills/jeremy-miner-identity-persuasion/SKILL.md`
   - `skills/jeremy-miner-identity-persuasion/genius.md` — All 28 patterns + all 10 hidden knowledge items
   - `skills/jeremy-miner-identity-persuasion/workflows/npq-conversation-architect.md`

2. **Gather inputs:**
   - Industry/niche
   - Target prospect (role, situation, pain level)
   - Offer (product/service/program)
   - Price range
   - Top 5 objections
   - Desired outcome for the prospect
   - Common problems in this niche

3. **Execute the NPQ Conversation Flow Architect** end-to-end

4. **Deliver:**
   - Complete 8-stage conversation script
   - Tonal annotations on every line
   - Verbal pacing markers
   - Objection prevention frames embedded at correct timing
   - Branching logic for different prospect responses
   - Industry-specific question customizations

## The 8 Stages

1. Opening — Nervous System Calming
2. Connection Questions — End result focus
3. Situation Questions — Current state mapping
4. Problem Awareness — 3-Layer Drill
5. Gap Amplification — Make gap unbearable
6. Identity & Objection Prevention — Deploy frames
7. Solution Bridge — Let them prescribe the solution
8. Commitment — Permission close
