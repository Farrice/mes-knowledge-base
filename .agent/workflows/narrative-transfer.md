# /narrative-transfer — Cross-Domain Narrative Transfer Engine

Take a story's gravitational skeleton from one domain and rebuild it in another — Pressfield's core superpower operationalized.

## Steps

1. Load the Steven Pressfield skill: `skills/steven-pressfield-narrative-mastery/SKILL.md`
2. Load genius context: `skills/steven-pressfield-narrative-mastery/genius.md`
3. Execute workflow: `skills/steven-pressfield-narrative-mastery/workflows/15-cross-domain-narrative-transfer.md`
4. Collect: source narrative + target domain + target audience + strategic intent
5. Follow all phases: Source Narrative X-Ray → Target Domain Mapping → Reconstruction → Adaptation Notes → Expert Stacking Recommendations
6. Run the Quality Gate at the end of the workflow
