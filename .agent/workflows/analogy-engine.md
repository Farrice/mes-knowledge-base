# Analogy Engineering

Make boring topics magnetic through strategic analogy mapping.

## Workflow

1. Load `skills/alex-content-science/genius.md`
2. Load `skills/alex-content-science/workflows/06-analogy-engineering.md`
3. Execute with user's boring topic, audience, and audience interest list
4. Deliver:
   - Boredom barrier diagnosis
   - Audience interest mapping with shared structures
   - 5+ analogy candidates with strength scores
   - Stress-tested winning analogy (simplifies, carries, shifts emotion, is original, respects audience)
   - Full content architecture built around the winning analogy
5. Quality gate: analogy carries full explanation, creates emotion shift, content titled through analogy not boring topic
