### Input
The user provides either:
- A story/script that needs an iconic moment engineered into it
- A theme or emotional payload that needs a signature line
- Content/copy that needs a "quotable" — something that lives beyond its context
- A request to generate memorable lines for any purpose (brand, speech, song, etc.)

### Step 1: Identify the Emotional Core
What is the deepest truth this moment needs to express? Not the surface message, but the thing that makes people go quiet.

### Step 2: Apply the Three Forge Methods

**Method 1 — The Metaphor Forge** (for signature lines)
- Start with the literal truth
- Find a concrete, physical metaphor from everyday life
- Ensure the metaphor works on multiple levels (literal, emotional, philosophical, universal)
- Test: Can a child understand it AND a philosopher find depth in it?

**Method 2 — The Visual Forge** (for iconic images)
- Identify the emotional climax of the scene
- Find a single, unrepeatable visual that captures the entire feeling
- The image should be simple enough to describe in one sentence but profound enough to think about for years
- Test: Can you close your eyes and see it? Would it work as a still photograph?

**Method 3 — The Scene Forge** (for unforgettable sequences)
- Build a complete scene architecture around one emotional revelation
- Use subtext, visual prose, and character psychology together
- The scene should feel inevitable in retrospect but surprising in the moment
- Test: Would someone describe this scene to a friend who hasn't seen the movie?

### Step 3: Generate 10+ Candidates
For each method, generate multiple candidates. Memorable moments aren't found on the first try — they're eroded into existence through iteration.

### Step 4: The Soul-Stay Test
For each candidate, ask:
- Will someone remember this in a year?
- Would someone quote this at a dinner party?
- Does it feel earned by the story, not manufactured?
- Does it reveal something true about the human condition?
