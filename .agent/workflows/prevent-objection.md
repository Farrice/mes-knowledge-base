# /prevent-objection

Stop objections before they form using identity frames, intention frames, necessity frames, and nervous system calming.

## Usage

```
/prevent-objection
```

## What It Does

Takes your top 3-5 objections and builds a complete prevention architecture with deployment timing, exact scripts, and written copy variants.

## Execution

1. **Read skill files:**
   - `skills/jeremy-miner-identity-persuasion/genius.md` — Focus on Patterns 20, 21, 22, 19, 15
   - `skills/jeremy-miner-identity-persuasion/workflows/objection-prevention-system.md`

2. **Gather inputs:**
   - What are your top 3-5 objections?
   - What context? (sales call, landing page, email, presentation)
   - What's your offer and price point?
   - Who is the typical prospect raising these objections?

3. **Execute the Objection Prevention System workflow** end-to-end

4. **Deliver:**
   - Objection inventory with identity analysis
   - Prevention architecture (which frame type for each)
   - Deployment timeline (when each prevention fires)
   - Exact scripts with tonal annotations
   - Written copy variants for landing pages / emails

## Stacks With

- `/npq-script` — embed prevention frames into a full conversation flow
- `/psych-trigger` — use triggers as the vehicle for prevention
- `/tone-calibrate` — ensure prevention frames have correct tonal delivery
- `/build-gap` — build a gap so wide that objections feel irrelevant
