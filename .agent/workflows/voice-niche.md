# /voice-niche — Voice Niche Positioner

Use the Voice Sensitivity Spectrum to analyze niches and position yourself where voice fidelity IS the product — commanding premium retainers and long-term retention.

## Usage

```
/voice-niche [describe your current experience and goals]
```

## Steps

### 1. Load the Skill
// turbo
Read `skills/sean-mabry-voice-mastery/genius.md`.
Read `skills/sean-mabry-voice-mastery/references/hidden-knowledge.md` — load Voice Sensitivity Spectrum insight.

### 2. Load the Prompt
// turbo
Read `skills/sean-mabry-voice-mastery/references/prompts/voice-niche-positioner.md`.

### 3. Gather Input
Ask the user:
- **Current experience**: Industries worked in, client types, content formats
- **Strengths**: What are you best at? (speed, depth, empathy, technical)
- **Revenue goals**: Monthly income target, desired client count
- **Current niche**: What you're doing now and what's working/not

### 4. Execute
1. Map current and candidate niches on Voice Sensitivity Spectrum
2. Score Niche-Voice Fit for 2-3 candidates
3. Craft niche positioning statement
4. Build 90-day transition plan (if needed)
5. Competitive moat assessment

### 5. Deliver
Present the Niche Strategy Brief with recommendations and transition plan.
