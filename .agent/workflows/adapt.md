### Input
The user provides source material — a book, article, true story, podcast, documentary, or any non-screenplay content to be adapted.

### Step 1: Source Material Autopsy
Read/analyze the source material and extract:
- **The Emotional DNA** — What is this story really about at its deepest level?
- **The Combustion Elements** — What 3-5 moments/characters/images are too good to lose?
- **The Disposables** — What works in the original medium but won't work on screen?

### Step 2: The Transmutation Decision
Determine the adaptation approach:
- **Faithful Translation** (10-30% invention) — Rare, only when source is already cinematic
- **Creative Transmutation** (40-60% invention) — Standard Roth approach
- **Radical Reinvention** (60-90% invention) — The Forrest Gump approach, where the source is a springboard

### Step 3: Theme Extraction
Run the Theme Compass Navigator on the source material. The theme you extract may be different from the original author's stated theme — that's fine. You're adapting, not translating.

### Step 4: Build the New Architecture
Using only the Combustion Elements + your new thematic compass:
- Design the five-act structure
- Build new characters, scenes, and subplots that serve YOUR theme
- Ensure at least 60% of the screenplay is original invention

### Step 5: The Fidelity Audit
Check each element against the source material:
- What was kept and why?
- What was invented and why?
- What was changed and what's the cinematic justification?
