Architect emotional value for any product or offer using Greg Hoffman's Nike methodology.

### PHASE 2: GATHER INPUTS

Ask the user for:
- **Product/offer/service**: What needs emotional architecture?
- **Current positioning**: How is it currently presented?
- **Audience**: Who uses this, and what do they aspire to?
- **Stories available**: Origin story, founder story, community stories, customer transformations?

If the user has already provided context, proceed directly.

### PHASE 3: EXECUTE THE WORKFLOW

Follow the Emotional Value Architect workflow exactly:
1. Emotional vs. Rational Value Audit (diagnose the ratio)
2. Identity Uniform Mapping (mentality, signal, community, aspiration)
3. Edition-Based Storytelling Blueprint (People, Place, Moment, Heritage editions)
4. Contrary Truth Content Engine (5 assumption→truth pairs)
5. Brand as Club Architecture (promise, tiers, privileges, creator pipeline)

### PHASE 4: QUALITY GATE

Before delivering, verify:
- Edition stories are rooted in reality (specific people, places, moments)
- Contrary Truth angles genuinely surprise
- Club architecture creates belonging, not just discounts
- Identity Uniform is specific enough to identify a brand "member" by behavior

A complete Emotional Value Architecture with Identity Uniform, Edition Blueprint, Contrary Truth angles, and Club Architecture.
