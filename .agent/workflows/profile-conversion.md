# LinkedIn Profile Conversion Blueprint

Turn your LinkedIn profile into a conversion machine using Josh Sanders' methodology.

## Workflow

1. Load `skills/josh-sanders-linkedin-growth/genius.md`
2. Load `skills/josh-sanders-linkedin-growth/workflows/01-profile-conversion-blueprint.md`
3. Execute with user's current profile, niche, and target audience
4. Deliver:
   - Profile audit (headline, banner, about, featured, experience)
   - Social proof tagline architecture
   - Visitor-to-follower conversion optimization
   - CTA placement strategy
5. Quality gate: profile reads as landing page not resume, social proof tagline is specific and verifiable
