# `/yt-flywheel` — The Deep Engine (v1.1)

This workflow builds a massive, long-form content ecosystem anchored by a YouTube script, atomized into social assets.

***This workflow enforces a strict Orchestrator model: The AI must generate a Deep-Dive Creative Brief and receive explicit user approval BEFORE writing the final 6-asset package.***

## When to Use
- User has a high-value concept (from `/flywheel-ideas`) ready for deep exploration.
- User is ready to record a talking-head video.

---

## Step 1: In-Loop Ideation (If starting from scratch)
If the user invokes `/yt-flywheel` *without* a specific topic or Mini-Brief:
1. Immediately run a 3-concept Ideation sprint (similar to `/flywheel-ideas`). 
2. Present the 3 options to the user.
3. **WAIT FOR USER APPROVAL** before proceeding to Step 2.

---

## Step 2: The Deep-Dive Creative Brief Generation (Mandatory)

Once a topic or Mini-Brief is selected, build the **Master Deep-Dive Brief**.

**The Brief must include:**
1. **The Core Hook & Stakes (Kallaway):** Why should the audience care immediately?
2. **The YouTube Architecture (Shaan Puri & PJ Accetturo):** 
   - Define the "5-Second Moment" (Low-Status Opening/Pattern Break).
   - Outline the "Intention + Obstacle" framing.
3. **The Thematic Arc (Mitch Albom):** The core emotional theme and central metaphor for the Substack article.
4. **The Vibe/Taste Check (Oren):** The intended aesthetic and tone.

**Present this Deep-Dive Brief to the user.**

> [!IMPORTANT]
> **HALT EXECUTION.** You must explicitly ask the user: "Does this Deep-Dive Brief look good? Do you want to tweak any of the Lego pieces or pacing before I draft the full 5-minute script and all the atomized assets?" Wait for their explicit GO.

---

## Step 3: Generate the Massive Asset Package

Only after the Deep-Dive Brief is approved, generate the final assets based strictly on the blueprint.

### Asset 1: YouTube Script (Puri & Accetturo)
- 5-8 minute script (approx 800-1200 words).
- Broken down by Timestamps and bracketed Visual Directions (e.g., `[Quick zoom on face]`).
- Staccato pacing for spoken delivery.

### Asset 2: Substack Article (Mitch Albom)
- 1,200 - 2,000 word thematic essay.
- Embeds the Prompt Kit.

### Asset 3: The Atomization Layer (Jasmin Alic & Lara Acosta)
- **3A: LinkedIn Long Article** (Condensed deep-dive, pulsing with authority).
- **3B: LinkedIn Short Post** (Trailer for the video, high contrast).
- **3C: LinkedIn Carousel Copy** (5-7 slide outline summarizing the core framework).

### Asset 4: Substack Note
- Rapid-fire, 2-sentence thought to hook subscribers.

### Asset 5: The Prompt Kit Lead Magnet (Oren)
- XML-tagged, multi-turn diagnostic. Link used in YouTube description.

### Asset 6: The Visual Asset Prompt Sheet (Sean Kochel)
- **YouTube Thumbnail:** 2 highly clickable visual concepts with specific text overlays.
- **Midjourney Prompts:** For B-Roll generation.
- **Canva Infographic Specs:** For the LinkedIn Short Post.
