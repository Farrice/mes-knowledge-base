# /tone-calibrate

Map Jeremy Miner's 5 tone types to any script or copy — annotate with tonal direction, pacing markers, and emotional flow.

## Usage

```
/tone-calibrate
```

## What It Does

Takes any existing script, sales copy, or content and annotates every line with the appropriate tone emoji, verbal pacing markers, and phase mapping.

## Execution

1. **Read skill files:**
   - `skills/jeremy-miner-identity-persuasion/genius.md` — Focus on Pattern 23 (5 Tone Types)
   - `skills/jeremy-miner-identity-persuasion/workflows/tonality-calibration-guide.md`

2. **Gather inputs:**
   - The script or copy to annotate
   - What context? (sales call, presentation, video script, written copy)
   - What's the primary goal? (build trust, create urgency, lower guard, close)

3. **Execute the Tonality & Verbal Pacing Calibration Guide** end-to-end

4. **Deliver:**
   - Fully annotated version with tone emojis (🔵🟡🔴🟢🟣)
   - Verbal pacing markers (..., [PAUSE], ↓, ↑)
   - Phase map showing tonal flow across the entire piece
   - Red flags where tonal misalignment would trigger fight-or-flight
   - Written tonal translation for copy (if written context)
