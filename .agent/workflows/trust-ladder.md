# /trust-ladder — Trust Ladder Navigator

Diagnose where a ghostwriting engagement sits on the trust progression, engineer the "no edits" milestone, and plan the path to irreplaceability.

## Usage

```
/trust-ladder [client name or engagement context]
```

## Steps

### 1. Load the Skill
// turbo
Read `skills/sean-mabry-voice-mastery/genius.md` — load the Trust Ladder Delegation pattern.

### 2. Load the Prompt
// turbo
Read `skills/sean-mabry-voice-mastery/references/prompts/trust-ladder-navigator.md`.

### 3. Gather Input
Ask the user:
- **Client**: Who is the client?
- **Engagement length**: How long have you been working together?
- **Current review process**: Do they review everything? Who publishes?
- **Edit patterns**: Heavy edits, light edits, or occasional no-edits?
- **Relationship dynamics**: Any friction points? Communication frequency?
- **Goal**: What should this relationship look like in 90 days?

### 4. Execute
1. Current stage diagnosis (Trial → Review → Trusted → Autonomous)
2. Advancement blocker identification
3. "No edits" engineering plan
4. Post-milestone strategy timeline
5. Irreplaceability score across 5 factors

### 5. Deliver
Present the Trust Ladder Assessment with action items.
