# /story-remix — Story Remix Engine

Take a client's story and deploy it across 4-6 formats (LinkedIn, email, book chapter, keynote, sales page) — each version emphasizes different elements for different audiences.

## Usage

```
/story-remix [paste the story or describe it]
```

## Steps

### 1. Load the Skill
// turbo
Read `skills/sean-mabry-voice-mastery/genius.md`.

### 2. Load the Prompt
// turbo
Read `skills/sean-mabry-voice-mastery/references/prompts/story-remix-engine.md`.

### 3. Gather Input
Ask the user:
- **Source story**: The original story (paste, reference, or describe)
- **Voice reference**: Voice Document or client content samples
- **Target formats**: Which platforms/formats need content?
- **Audience variants**: Different audience segments to target?

### 4. Execute
1. Story anatomy breakdown (setup, inciting moment, struggle, dilemma, lesson, detail, emotional core)
2. Remix matrix — which elements lead for each format
3. Draft each remix in the client's voice
4. Freshness check — ensure no version feels like a repeat

### 5. Deliver
Present the Story Remix Package with drafts for each format.
