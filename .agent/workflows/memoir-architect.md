# /memoir-architect — Memoir Architect

Help a thought leader position and structure their first book using the Authorship Pyramid model and Dan Harmon's Story Circle for narrative architecture.

## Usage

```
/memoir-architect [client name or context]
```

## Steps

### 1. Load the Skill
// turbo
Read `skills/sean-mabry-voice-mastery/genius.md` — load the Authorship Pyramid pattern.

### 2. Load the Prompt
// turbo
Read `skills/sean-mabry-voice-mastery/references/prompts/memoir-architect.md`.

### 3. Load Implementation Reference
// turbo
Read `skills/sean-mabry-voice-mastery/references/implementation.md` — the Memoir Book Architecture methodology.

### 4. Gather Input
Ask the user:
- **Client**: Who is the author?
- **Industry/niche**: What space are they in?
- **Goal**: What should the book accomplish? (authority, leads, legacy, platform)
- **Existing content**: Interview material, Hidden Gems bank, existing stories?
- **Book type preference**: Memoir, tactical how-to, or hybrid?

### 5. Execute
1. Industry pyramid mapping (top 0.01%, middle practitioners, base dreamers)
2. Book type recommendation with reasoning
3. Memoir scoping — 3-5 chapters of the client's life that support the thesis
4. Book proposal outline with flexibility markers
5. Hero's journey architecture per chapter

### 6. Deliver
Present the Book Blueprint with positioning rationale, chapter outline, and interview plan.
Recommend: "Run `/client-interview book` for structured material extraction."
