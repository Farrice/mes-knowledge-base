# /design-digital-product-offer — Design Digital Product Offer

End-to-end pipeline that takes you from raw idea to revenue-generating digital offer. Chains Maria Wendt's research-first methodology with outcome selling, strategic pricing, and trust-based marketing.

## Usage

```
/design-digital-product-offer [describe your idea, expertise, or the problem you want to solve]
```

## Pipeline

### Phase 1: Research & Validation
Deploy **Workflow 05: Research-First Product Validator**
- Mine buyer language from Reddit, Facebook, Amazon reviews
- Validate demand across 3+ channels
- Generate 5 product concepts, ranked by evidence
- Produce outcome-positioned headline and one-line pitch

**OUTPUT**: Validated Product Concept Document

### Phase 2: Price & Launch Plan
Deploy **Workflow 06: Strategic Price & Launch Engine**
- Calculate the problem's economic value
- Set price at 5-25% of value (obvious steal)
- Scope MVP (create in 1-5 days max)
- Generate 7-Day Launch Sprint plan

**OUTPUT**: Price Architecture + MVP Spec + Launch Sprint

### Phase 3: Sales Page Optimization
Deploy **Workflow 09: Sales Page & Messaging Optimizer**
- Generate 20 headline variations, rank top 5
- Build sales page blueprint (Four Answers Block)
- Design testimonial collection strategy
- Add guarantee and real urgency (if applicable)

**OUTPUT**: Sales Page Blueprint + Messaging Kit

### Phase 4: Audience & Trust Strategy
Deploy **Workflow 07: Trust-First Audience Engine**
- Sharpen audience to 90% exclusion
- Build 9:1 trust content ratio plan
- Generate 30-day trust content calendar
- Design conversion mechanics (natural byproduct, not hard sell)

**OUTPUT**: Trust-First Audience Strategy

### Phase 5 (Optional): First Sale Sprint
If starting from absolute zero, deploy **Workflow 08: First Sale Sprint**
- Create $5-$10 micro-product in 60 minutes
- Set up checkout with free tools
- Execute Find-One-Help-One direct outreach
- Get first paying customer within 48 hours

**OUTPUT**: First Sale Action Plan

## Quick Start Options

- **"I have an idea but haven't validated"** → Start at Phase 1
- **"I have a product but it's not selling"** → Start at Phase 3
- **"I want my first sale ever"** → Skip to Phase 5
- **"Full pipeline, end to end"** → Run Phases 1→2→3→4
