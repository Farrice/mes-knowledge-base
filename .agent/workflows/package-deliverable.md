# /package-deliverable — Premium Client Deliverable Packaging

> **Why this exists:** Our strategy and content work is world-class, but raw markdown files don't communicate that to premium clients. This workflow bridges the gap between great content and great presentation.

## The Packaging Gap

We produce exceptional strategic documents:
- Brand identity systems
- Positioning audits
- Avatar dossiers
- Content playbooks
- Funnel architectures

But delivering them as .md or .txt files undersells the work. Premium clients expect premium packaging.

## Packaging Options (Ranked by Polish)

### Tier 1: Quick & Professional (10 min per doc)
**Tool:** Gamma.app (AI-powered presentations)
- Paste markdown content → Gamma auto-designs slides/docs
- Good for: Strategy decks, playbooks, one-pagers
- Output: Shareable link or PDF export
- Cost: Free tier available, Pro ~$10/mo

### Tier 2: Branded & Beautiful (30 min per doc)
**Tool:** Canva / Beautiful.ai
- Use brand templates with locked colors, fonts, logos
- Good for: Client-facing deliverables, proposals, case studies
- Output: PDF, branded deck
- Setup: Create a brand kit once, reuse across all clients

### Tier 3: Maximum Premium (1-2 hours per package)
**Tool:** Custom HTML → PDF (using our own design system)
- Build a branded HTML template with print CSS
- Pipe markdown content through it
- Good for: Full engagement packages, "the binder" experience
- Output: Beautifully typeset PDF that looks like a $50K consulting report

### Tier 4: Interactive Experience
**Tool:** Notion / Gamma / Custom site
- Deliver as an interactive, navigable digital experience
- Good for: Ongoing engagements, living documents
- Output: Private link with embedded media

## Recommended Workflow for Client Packages

1. **Finish all content work** in markdown (strategy, copy, positioning, etc.)
2. **Run a quality pass** — `@oren` CEV critique + `@harry-dry` concrete copy check
3. **Choose packaging tier** based on client level and budget
4. **Build the visual package:**
   - Export content sections to chosen tool
   - Apply brand template (client's brand OR your agency brand)
   - Add visual hierarchy: headers, callouts, pull quotes, icons
   - Include data visualizations where relevant
5. **Final review** — Does this look like something a $5K-$10K engagement would produce?
