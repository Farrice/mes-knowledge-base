# /build-gap

Map the gap between where your prospect IS and where they COULD BE — the wider the gap, the stronger the urgency.

## Usage

```
/build-gap
```

## What It Does

Takes a prospect profile or audience description and generates a complete gap architecture: current state mapping, objective state mapping, 3-layer problem drill questions, gap amplification sequences, and self-persuasion bridge questions.

## Execution

1. **Read skill files:**
   - `skills/jeremy-miner-identity-persuasion/genius.md` — Focus on Patterns 25, 26, 27, 24
   - `skills/jeremy-miner-identity-persuasion/workflows/gap-building-engine.md`

2. **Gather inputs:**
   - Who is the prospect? (role, industry, situation)
   - What's their likely current state?
   - What's the objective state your solution enables?
   - What context? (sales script, landing page, content, email)

3. **Execute the Gap Building Engine workflow** end-to-end

4. **Deliver:**
   - Current State Map (in prospect's language)
   - Objective State Map (vivid, emotional)
   - 3-5 problems with full 3-Layer Drill (real problem → root cause → personal impact)
   - Gap amplification sequence
   - Self-persuasion bridge questions

## Stacks With

- `/psych-trigger` — apply psychological triggers after building the gap
- `/npq-script` — embed gap building into a full conversation flow
- `/self-persuade` — design questions that make them articulate the gap
- `/prevent-objection` — prevent objections that arise from an inflated gap
