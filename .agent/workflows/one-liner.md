## PHASE 2: GATHER INPUTS

Ask the user for:
1. **Business name** and what it sells
2. **Target customer**
3. **Primary problem solved**
4. **Existing BrandScript** (optional — if available, extract from it)

## PHASE 3: EXECUTE THE WORKFLOW

Follow the `04-one-liner-generator.md` workflow exactly:
1. Problem Identification (most emotionally resonant problem)
2. Solution Mechanism (what the business does, not product name)
3. Result Painting (aspirational outcome)
4. Assemble the One-Liner (5 variations: standard, conversation, social bio, email signature, pitch)
5. Stress Test (Taxi Test, Repeat Test, Tell Me More Test, Survival Test)

## PHASE 4: QUALITY GATE

Before delivering, verify:
- [ ] Problem triggers customer recognition ("that's me")
- [ ] Solution implies competence without jargon
- [ ] Result creates genuine desire
- [ ] Passes all 4 stress tests
- [ ] Deployable today across 5+ channels

## OUTPUT

Core one-liner + 4 variants + deployment guide showing exactly where to use each version.
