# Content Remix Lab

Generate original content by applying extracted principles to your niche.

## Workflow

1. Load `skills/alex-content-science/genius.md`
2. Load `skills/alex-content-science/workflows/03-content-remix-lab.md`
3. Execute with extracted principles + user's niche, audience, and platform
4. Deliver:
   - Principle-to-niche translations (3 applications per principle)
   - Cross-pollinated concept combinations
   - Originality checks (Purple Cow, Source Trace, Voice, Refusal)
   - Full content briefs with Detail Stack decisions
   - Priority ranking
5. Quality gate: all concepts pass Purple Cow + Source Trace tests, 3+ Detail Stack decisions per concept
